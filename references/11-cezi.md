# 测字 (Glyphomancy / 字占)

## 一、测字概述

测字, 又称"破字""相字""拆字", 是以汉字之形、音、义、笔画为媒介, 推断所问之事吉凶的术数。其本质为以汉字承载的丰富信息, 结合问事者的情境与心念, 作出针对性的应答。

### 源流

- 测字可追溯至上古"卜字"传统, 汉字本身从象形而来, 字字含义。
- 唐宋时期渐成体系, 与卦理结合。
- **谢石**: 南宋测字名家, 留下多则著名公案, 被视为测字宗师。
- **邵雍 (邵康节)**: 北宋理学家、易学大家, 其测字融《易》理与字理, 玄妙莫测。
- 明清时期测字盛行, 与街边江湖术结合, 出现专门测字摊。

### 测字与汉字的关系

汉字六书 (象形、指事、会意、形声、转注、假借) 是测字的语言学基础。汉字每个偏旁、笔画都承载意义信息, 为测字提供了广阔的解读空间。

---

## 二、测字基本原理

测字者依据字的多维度信息综合判断:

### 1. 字形结构

- **字的整体形状**: 方圆、长扁、上下结构、左右结构、包围结构。
- **偏旁部首**: 各偏旁承载的本义 (如氵主水、火主热、人主人事)。
- **形象联想**: 字形所象之物 (如"中"如旗, "卜"如人立)。

### 2. 字音

- **谐音**: 同音字的双关意义。
- **声调**: 平仄变化中的情绪信息。
- **方言读音**: 部分情境下方言读音亦可解读。

### 3. 字义

- **本义**: 字的原始含义。
- **引申义**: 字在历史演变中的扩展义。
- **典故义**: 字在经典文献中的特殊含义。

### 4. 笔画数

- 笔画的奇偶 (奇为阳, 偶为阴)。
- 笔画总数对应的五行 (尾数法)。
- 笔画数转卦数。

### 5. 五行属性

- 偏旁五行 (氵水, 火, 木, 金, 土)。
- 笔画五行。

## 三、测字技法

测字非死法, 高手随机应变, 但有基本技法可循:

### 1. 拆字 (Split)

将字拆分为多个部件, 分别取义。

**例**: "好"字 = "女" + "子"。可解为"有女子相伴则好", "女子之事吉"。

**例**: "李"字 = "木" + "子"。可问"是否与木姓人合作?", 或"是否得子?"。

### 2. 添字 (Add)

在字上添加笔画或偏旁, 看变成何字, 推断走向。

**例**: "口"字添一画可成"日""中""目""可""叶"等, 各示不同走向。

### 3. 减字 (Subtract)

从字上减去笔画或偏旁, 看变成何字。

**例**: "亲"字减去"立"为"亲"的下部, 减去后未必成字, 但缺失意义可推断关系破损。

**例**: "天"字减去一横为"夫"; 再减为"大"; 再减为"人"。可推断地位变化。

### 4. 添笔 / 减笔

仅一笔之差, 字义全变。

**例**: "甲"字加一竖为"申"; 加一横为"由"; 加一点为"申"或"电"。

### 5. 反字 / 倒字

将字反转或倒立, 寓意未来逆转或当下颠倒。

**例**: "由"倒过来为"甲" (古字); "末"与"未"差一字, 反映方向相反。

### 6. 谐音

利用同音字进行解读。

**例**: 问健康写"洋"字, 谐音"扬", 主病情向上扬; 但若与水有关, 主病及肾。

**例**: 写"梨"字问婚姻, 谐音"离", 主分离 (古例)。

### 7. 字象

字的整体形象引发联想。

**例**: "门"字, 象一座门, 可解为"事有出入""须等开门时机"。

### 8. 字意

直接取字本义。

**例**: 写"忧"字, 本义为愁苦, 直接对应当事人状态。

## 四、起字方法

测字的"起字"方式多样, 起字方式本身已蕴含信息:

### 1. 来人写字

让来人随意写一字, 此字最关键。
- 写字时的笔画粗细、墨色、字迹工整与否, 均可参考。
- 写字时的心境通过笔触流露。

### 2. 来人报字

让来人口报一字。
- 报字的语速、声调、是否犹豫均为信息。
- 报字本身已是潜意识选择。

### 3. 抓阄

在多个字中随机抽一字, 凭"心动"取字。

### 4. 翻书

随手翻一书页, 取首字或眼缘字。

### 5. 据时取字

按当下时辰、地点、情境取字。

### 6. 偶遇字

街上偶见广告牌、招牌某字, 即取此字。

### 重要原则

起字方法不同, 但都遵循一个原则: **意念所至, 字必应之**。来人的心念已暗中决定了所取之字。

## 五、应事原则

### 1. 同时考虑情境

测字非纯字论字, 必须结合:

- **来人神色**: 焦虑、平静、犹豫各示不同。
- **所问之事**: 问婚、问财、问病, 同字答异。
- **时令时辰**: 春测字偏木, 冬测字偏水。
- **来人身份**: 商人、官员、学生, 同字解异。
- **当时环境**: 噪杂、安静、突变各为参照。

### 2. 字配卦

将字归入八卦, 借卦理解读:

| 卦 | 卦象 | 对应字类 |
|----|------|---------|
| 乾 | 天 | "天""君""父""金""玉""王""首"等 |
| 坤 | 地 | "地""母""牛""布""釜""土""黄"等 |
| 震 | 雷 | "雷""动""龙""木""声""惊""车"等 |
| 巽 | 风 | "风""木""绳""鸡""股""花""草"等 |
| 坎 | 水 | "水""月""血""车""陷""鱼""智"等 |
| 离 | 火 | "火""日""目""文""明""鸟""红"等 |
| 艮 | 山 | "山""石""门""径""手""狗""少男"等 |
| 兑 | 泽 | "泽""口""毁""折""少女""巫""金"等 |

### 3. 时机原则

测字所应之事, 通常应于:
- **当下三日内**: 紧急之事。
- **当月内**: 一般之事。
- **三月内**: 长远之事。

时间长则变量多, 不宜断得太远。

## 六、经典案例

### 案例一: 谢石测"杭"字 (宋高宗故事)

宋高宗曾召谢石入宫, 写一"杭"字让其测。

**谢石解**: "'杭'字, 木边亢也。陛下当兴于杭州, 然'亢'字上有点, 中有横, 是'亢龙有悔'之象。陛下虽兴于此地, 然偏安一隅, 北望中原, 终是悔多于喜。"

后高宗果定都临安 (杭州), 偏安南宋, 终未能恢复中原。

**评析**: 此例展示了**拆字 + 卦理 + 历史背景**结合的高级测法。

### 案例二: 谢石测"问"字

一妇人写"问"字, 问出门之夫归期。

**谢石解**: "'问'字, 门中有口。门, 出入之处; 口, 食也。今日傍晚君必至门, 共同进食。"

果然当日傍晚, 其夫归来。

**评析**: 字形直观联想 + 时机把握。

### 案例三: 邵雍测"亡"字

邵康节路遇一人, 见其写"亡"字。

**邵雍解**: "'亡'字, 上有点, 中有一, 下有'乚'。点为头, 一为颈, 乚为蜷曲之身。今日必有人头部受伤, 卧床休养。"

果然当日有伤者被抬过。

**评析**: 此为"字象"高度敏感的运用, 偏重直觉判断。

### 案例四: 测"也"字

一人写"也"字, 问婚事是否成。

**测字者拆解**: "'也'字, 加'女'为'她', 表示有女子; 加'氵'为'池', 表示池中之水, 浅薄; 加'马'为'驰', 表示奔忙; 加'木'为'杝', 木相疏。"

**结论**: 此事女子方面有意, 然过程浅薄、奔忙、各方不齐。终结果有缘无份。

**评析**: 此例为"添字"技法的典型, 展示一字多解之妙。

### 案例五: 测"乾"字

宋时一举子赴考前, 抓阄得"乾"字。

**测字者解**: "'乾'字, 卦为乾天, 大吉。然'乾'字右半 '乞' 也, 是有所求; 左半 '倝' 字罕用, 主孤高。君此次必中, 但中后须谦逊, 否则有'乞讨'之嫌。"

后此举子果然中举, 然初任傲慢, 遭弹劾, 应了"乞"之兆。

## 七、字配五行

### 偏旁部首五行

| 五行 | 偏旁部首 |
|------|---------|
| 木 | 木、艹 (草)、竹、禾、米 (部分)、林、森 |
| 火 | 火、灬 (四点底)、日、光、赤、丹、夏、灯、烈、煌 |
| 土 | 土、田、山、石、阜 (左阝)、邑 (右阝, 部分)、岛 |
| 金 | 钅 (金)、刀、刂、戈、斧、铜、铁、刚、剑 |
| 水 | 氵、冫、雨、川、月 (部分)、淼、雪、霜、海 |

**特别说明**:
- **阝 (左)**: 阜部, 与山、土、阶梯相关, 偏土。
- **阝 (右)**: 邑部, 与城邑、地域相关, 偏土。
- **月**: 部分字源为"月" (水), 部分为"肉" (土)。

### 笔画数五行 (尾数法)

| 笔画尾数 | 五行 |
|---------|------|
| 1、2 | 木 |
| 3、4 | 火 |
| 5、6 | 土 |
| 7、8 | 金 |
| 9、0 | 水 |

**例**: "李"字7画, 尾数7, 属金。
**例**: "张"字繁体11画, 简体7画 (依繁体), 尾数1, 属木。

注: 笔画法在不同流派略有差异, 有的用康熙字典笔画, 有的用现代规范笔画。

### 字配五行用法

- 测字时, 看所测字之五行与来人八字五行是否相合。
- 相生则吉, 相克则凶。
- 用神所喜之五行字, 主该方面顺。

## 八、测字与心理

现代心理学视角下, 测字可解读为一种**投射测验** (Projective Test), 类似罗夏墨迹测验:

- 来人选何字、写何字, 反映其当下潜意识关注。
- 字的笔画轻重缓急, 反映书写者情绪状态。
- 测字者的"解"实际上是结合来人状态进行的针对性引导。

### 测字的"灵验"机制

1. **巴纳姆效应**: 模糊语句让人觉得贴合自身。
2. **冷读术 (Cold Reading)**: 通过观察来人的细微反应调整解读。
3. **潜意识投射**: 来人选字本身已反映其内心倾向。
4. **自我应验预言**: 解读影响来人行为, 使预言成真。
5. **概率与情境**: 多数测字应答涵盖范围广, 易找到对应。

### 测字的现代价值

去除迷信成分后, 测字仍有以下价值:

- **自我反思工具**: 通过随意写字, 反观当下心境。
- **咨询沟通切入**: 作为打开对话的媒介, 帮助来人表达困扰。
- **创意启发**: 字的多重解读能激发联想。
- **传统文化体验**: 汉字之美与术数之趣。
