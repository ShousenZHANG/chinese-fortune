# 面相 (Physiognomy / 看相)

## 一、面相概述

面相, 又称相术、相人之学, 是中国传统术数中以人之面部形貌、骨骼、气色、神态推断其性格、命运、健康、际遇的一门学问。其源流可追溯至先秦, 《左传》《国语》中已有"相人"之载, 至汉魏六朝渐成体系, 宋元以后蔚为大宗。

### 经典源流

- 《麻衣相法》: 五代时期麻衣道者所传, 后世相术之祖, 偏重相骨、相神。
- 《柳庄相法》: 明代袁忠彻 (袁柳庄) 所著, 体系完备, 重实证, 多帝王将相案例。
- 《冰鉴》: 清代曾国藩所撰 (一说托名), 重在识人用人, 偏向"神骨"层面, 对官场及现代人事甄选影响深远。
- 《神相全编》: 集前代相书大成。
- 《公笃相法》: 民国陈公笃所著, 近代相术集大成之作。

### 心相 > 面相

相术第一原则: **相由心生, 心相重于面相**。

《麻衣》云:

> "未观相貌, 先看心田; 有相无心, 相随心灭; 无相有心, 相由心生。"

此为相术总纲: 面相只是当下状态的反映, 而内心修为可改变面相。古德常言"相不独论", 凡相必兼神、骨、气、声、行, 不可仅凭一处便下断语。**相之大忌, 在于一望即断、武断恐吓**。

故现代解读面相, 当持"参考性观察, 非命定论断"之心态。

## 二、三停 (三个区域划分)

将面部纵分为三等份, 称为"三停":

| 区域 | 范围 | 主管 | 主何事 |
|------|------|------|--------|
| 上停 | 发际线至眉毛 (天庭) | 早年 (15–30岁) | 智慧、父母荫庇、出身、学业、思考力 |
| 中停 | 眉毛至鼻底 (中庭) | 中年 (31–50岁) | 事业、夫妻、自我意志、社交、决断力 |
| 下停 | 人中至下颌 (地阁) | 晚年 (51岁后) | 子女、下属、福禄、晚境、田宅根基 |

### 三停均匀

理想为三停长度均匀, 比例约 1:1:1。**三停均匀者, 一生平顺**。

- 上停长而中下停短: 早年优势, 中晚乏力, 思想丰富但执行不足。
- 中停长而上下停短: 中年事业旺, 易自我意志强、固执。
- 下停长而上中停短: 晚景殷实, 早年辛苦。

### 三停各自的相理

- **上停 (天庭)**: 宜饱满、宽广、明润, 忌窄、塌、纹乱。天庭饱满者, 主聪明、贵气、父母有荫。
- **中停 (中庭)**: 以鼻为主, 兼看颧、眉、眼。宜端正、饱满、有气势, 忌偏斜、塌陷。中停正者, 主中年事业稳。
- **下停 (地阁)**: 宜方圆、丰厚、有肉, 忌尖、薄、削。地阁丰隆者, 主晚境福寿。

## 三、五官详解

五官者: 眉、眼、鼻、口、耳。古谓"五官各管十年", 然实为终身参看。

### 1. 眉 (情谊 / 性格)

眉为"保寿官", 又称"兄弟宫", 主情谊、性格、寿元根基。

- **眉宜清秀**: 毛顺、有彩、长过目者, 主聪明有情。
- **眉忌散乱**: 短促、稀疏、逆乱者, 主刑伤兄弟、性情急躁。
- **眉间宽**: 印堂广阔者, 心胸宽广。
- **眉间窄**: 双眉相连者, 主小气、心事重、早年困顿。
- **眉浓**: 主血气旺、感情厚, 然亦易冲动。
- **眉淡**: 主理智、寡情, 亦主兄弟缘薄。

### 2. 眼 (心灵 / 寿命)

眼为"监察官", 神之所聚, 心之苗。

> "观人之相先观神, 观神先观目。"

- **眼宜含藏**: 神光内敛者, 主寿、贵、聪慧。
- **眼忌浮露**: 三白眼 (上、下、左右过白) 者, 主刑克、刚烈。
- **眼忌泪堂下陷**: 主子女缘薄。
- **黑白分明**: 心地清明, 思虑清晰。
- **目光呆滞或飘忽**: 心神不定, 健康有亏。

### 3. 鼻 (财帛 / 自我)

鼻为"审辨官", 又称"财帛宫", 居面之中央, 主自我、财禄、夫妻。

- **鼻宜端正、有肉、有梁**: 自我意志坚定, 财运稳健。
- **鼻头有肉、鼻翼丰厚 (兰廷)**: 主财库厚实, 储蓄有方。
- **鼻梁低塌**: 主自我弱、依赖性强、中年运薄。
- **鼻孔仰露**: 主漏财、不留余。
- **鼻歪斜**: 主婚姻波折、中年是非。

### 4. 口 (信誉 / 出纳)

口为"出纳官", 主信誉、口福、晚景。

- **口宜方正、有棱角、唇红齿白**: 主信用、有口福、晚景顺。
- **口忌歪斜、薄削、唇紫黑**: 主信用缺、刑克。
- **口大有收**: 口大而能闭合者, 主威权。
- **口小**: 主谨慎、口紧, 然亦主格局小。
- **唇厚**: 主忠厚、情重。
- **唇薄**: 主薄情、利口。

### 5. 耳 (聪明 / 寿元)

耳为"采听官", 主早年、寿元、肾气。

- **耳宜高过眉、轮廓分明、贴脑、色润**: 主聪慧、早慧、寿长、根基厚。
- **耳忌反、低、薄、暗**: 主早年艰辛、根基弱。
- **耳垂厚**: 主有福、有寿。
- **耳垂尖小**: 主福薄。
- **左耳缺**: 主父先损; **右耳缺**: 主母先损 (古说, 参考性)。

## 四、十二宫

将面部划分为十二个区域, 各主一事, 称"十二宫"。

| 宫位 | 位置 | 主管事项 |
|------|------|---------|
| 命宫 | 两眉之间印堂 | 一生根基、近期运势、心智状态 |
| 财帛宫 | 鼻 (含准头、兰廷) | 财运、积蓄、自我经济能力 |
| 兄弟宫 | 双眉 | 兄弟姐妹、朋友、情谊 |
| 田宅宫 | 眉眼之间 (上眼睑) | 不动产、家业、田园 |
| 男女宫 | 眼下卧蚕 (泪堂) | 子女缘、生育、感情 |
| 奴仆宫 | 下颌两侧 (地库、颐) | 下属、晚辈、晚年田宅 |
| 妻妾宫 | 眼尾 (鱼尾、奸门) | 婚姻、配偶、夫妻感情 |
| 疾厄宫 | 山根 (两眼之间鼻梁起点) | 健康、灾厄、抗压能力 |
| 迁移宫 | 额角两侧 (驿马位、天仓) | 远行、迁徙、外地发展 |
| 官禄宫 | 额头中正 (司空) | 事业、官运、社会地位 |
| 福德宫 | 眉上天仓部位 | 福分、修养、内心安宁 |
| 父母宫 | 日月角 (额上两侧) | 父母关系、长辈缘 |

### 十二宫看相要点

- 每宫须"丰满、明润、无纹无痣无伤"为佳。
- 命宫 (印堂) 须明亮: 主近期顺遂; 印堂发暗则近期不利。
- 财帛宫 (鼻) 须丰厚: 财库充实。
- 妻妾宫 (鱼尾) 忌纹路杂乱: 主婚姻波折。
- 疾厄宫 (山根) 忌断、横纹: 主健康有亏。

## 五、五形脸

依据中国传统五行学说, 将面型分为五种, 各对应五行属性:

| 形 | 五行 | 特征 | 性格倾向 |
|----|------|------|---------|
| 木形 | 木 | 清瘦高挑, 面长方颀, 骨节凸显 | 仁慈、清雅、思虑细腻, 易神经质 |
| 火形 | 火 | 尖瘦红润, 面带尖角, 颧露 | 急智、热情、刚烈, 易冲动 |
| 土形 | 土 | 敦厚方正, 肉厚骨重, 色黄 | 厚重、诚信、稳实, 略迟缓 |
| 金形 | 金 | 方正白净, 面方棱正, 色白 | 刚毅、果断、有原则, 略冷峻 |
| 水形 | 水 | 圆胖丰润, 面圆肉厚, 色黑 | 圆融、智慧、变通, 易享乐 |

### 五形配五行运

- 看五形, 须看是否"得形"。例如水形人, 须圆胖丰润才算得形; 若水形人偏瘦削, 则为"形不配", 主不顺。
- 五形与年份五行相合, 主当年顺遂; 相克, 主当年破败。
- **混合形**最为常见, 单一形较少。

## 六、痣相

痣者, 黑子也, 古人视为身体上的"印记", 各有吉凶。

### 显痣与隐痣

- **显痣**: 长在面、手、颈等外露部位。古云"显痣多凶, 隐痣多吉"。
- **隐痣**: 长在衣物覆盖处 (胸、背、腰、臀、足心)。古云"隐痣多吉"。

### 好痣与恶痣判断

| 标准 | 好痣 | 恶痣 |
|------|------|------|
| 色 | 漆黑、朱红、纯白 | 灰、褐、杂色 |
| 形 | 圆润饱满 | 边缘不规则、扁平 |
| 光 | 光泽明润 | 暗淡无光 |
| 大小 | 适中, 凸出有气 | 过小如尘, 或过大不正 |
| 毛 | 长毛者尤吉 (称"长寿毛") | 无毛或毛色杂 |

### 不同位置痣相含义 (简表)

| 位置 | 含义 (传统说法, 仅供参考) |
|------|---------|
| 额上中央 | 早年贵气, 头脑灵活 |
| 印堂 | 大智大福, 然亦主孤高 |
| 眉中藏痣 | 才智过人, 草里藏珠 |
| 眼下 (泪堂) | 主泪痣, 多情或子女操心 |
| 鼻头 | 财来财去, 易破财 |
| 鼻翼 | 漏财 |
| 人中 | 子女缘佳 (清晰深长者) |
| 嘴角 | 食禄丰、口才好 |
| 下巴 | 田宅、晚景 |
| 耳内 | 长寿、福气 |
| 耳垂 | 财气 |
| 颈后 | "苦情痣" — 民俗说法 |
| 胸口正中 | 心怀大志 |
| 背上 | "担子痣", 主肩担家业 |

### 现代医学警告

**重要提示**: 现代医学已确认部分痣存在恶变可能 (黑色素瘤等)。凡符合 "ABCDE" 标准的痣应及时就医:

- **A**symmetry (不对称)
- **B**order (边缘不规则)
- **C**olor (颜色不均, 杂色)
- **D**iameter (直径 > 6mm)
- **E**volving (近期变化: 增大、出血、瘙痒)

**健康优先**: 任何疑似异常的痣, 不论吉凶相理如何, 都应优先咨询皮肤科医生。相术不能替代医学。

## 八、气色

气色是相术中最复杂、最难掌握的部分, 指面部当下显现的颜色、光泽、神韵。

- **气**: 看不见但能感觉到的精神状态、能量。
- **色**: 面部皮肤呈现的色泽。

### 五种气色

| 气色 | 主何事 |
|------|--------|
| 青 | 忧愁、惊吓、肝郁 |
| 红 | 喜事、桃花、口舌, 太过则火气大 |
| 黄 | 平顺、即将得财、土气旺 |
| 白 | 哀伤、孝服、肺虚 |
| 黑 | 灾厄、肾亏、近期不顺 |

### 看气色的时机

气色主**近期 (一周至三月内)** 运势, 与流年、流月、流日紧密相关。气色非永久, 一日数变, 须当面观察方准。古高手能预测三日之事, 即靠气色。

## 十三、现代视角

面相学的本质, 可视为**对人类生理特征与命运/性格关联的大规模经验归纳**, 与现代心理学的"薄片判断 (thin-slicing)"有相通之处。

### 薄片判断 (Thin-Slicing)

社会心理学家 Nalini Ambady 等人的研究表明, 人在数秒内通过面部线索就能对他人性格、能力做出与长期观察相近的判断, 准确率显著高于随机水平。这意味着:

- 面部特征确实承载性格信息 (荷尔蒙水平、骨骼发育、表情习惯长期塑造肌肉与皱纹)。
- 长期情绪状态会塑造面部 ("怒目"会形成习惯性肌肉记忆; "笑口常开"形成上扬鱼尾纹)。
- 因此"相由心生"在现代神经心理学上有部分实证依据。

### 局限与误区

- 面相非命定论。人的命运是基因、环境、选择的复合产物, 面相只反映其中部分线索。
- 古代相书中部分论断 (如某痣克夫、某眉克子) 缺乏实证, 不宜机械套用。
- 不应以面相为基础对他人进行歧视、用人决策、婚配选择。
- 相术应作为**自我观察、识人参考**, 而非**断人前程**的工具。

### 心相 > 面相 重申

修心可改相, 古今皆有例证。曾国藩晚年曾自述, 30岁前后面相变化巨大, 与其修身养性的功夫直接相关。**与其忧虑面相吉凶, 不如修心养性, 自然显于面**。
