# 手相 (Palmistry / 掌纹学)

## 一、手相概述

手相, 又称掌相、掌纹学, 是通过观察手掌的纹路、形状、丘陵、指节、指甲、气色等推断人之性格、健康、运势的术数。中国手相源出相术, 与面相同属"形相"系统; 西方手相学 (Palmistry, Chiromancy) 则独立发展, 流传于古希腊、印度、罗马, 近代以爱尔兰 Cheiro (基罗) 集大成。

### 中西手相异同

| 项目 | 中国手相 | 西方手相 |
|------|---------|---------|
| 理论基础 | 阴阳五行、八卦、易理 | 占星学、行星对应 |
| 主要内容 | 八卦定位、十二宫、流年 | 七大丘 + 主要线 |
| 与命理关系 | 与八字、面相结合 | 与占星结合 |
| 代表 | 《柳庄相法·手相篇》、《神相全编》 | Cheiro《掌相奥秘》 |

二者在主要纹路 (生命线、智慧线、感情线) 的解读上**高度相似**, 反映了人类对手掌特征的共同观察。

## 二、左右手

### 传统观: 男左女右

- 古云"男左女右": 看男子以左手为主, 女子以右手为主。
- 缘由: 阴阳学说中, 男属阳 (左), 女属阴 (右)。

### 现代观: 左手先天, 右手后天

- **左手**: 反映先天禀赋、潜在能力、命中带来的素质 (基因、本性)。
- **右手**: 反映后天发展、现实运势、实际际遇 (修为、选择、环境)。
- 看相时**两手对比**:
  - 左手好右手差: 先天好但浪费, 未发挥潜能。
  - 左手差右手好: 后天修为弥补, 励志改命。
  - 两手相同: 命运按既定轨道走。
  - 两手差异巨大: 人生有重大转折点。

### 实际操作

左撇子与右撇子: 现代以惯用手为"现实手 (后天)", 非惯用手为"潜在手 (先天)"。

## 三、八卦在手 (掌中八卦定位)

中国手相独有的体系, 将八卦方位映射到手掌:

```
        离宫
   巽宫        坤宫
震宫    明堂    兑宫
   艮宫        乾宫
        坎宫
```

具体位置:

| 卦 | 位置 | 主何事 |
|----|------|--------|
| 乾宫 | 大拇指根部 (拇指掌侧厚肉处, 即金星丘) | 父亲、长辈、贵人、官禄 |
| 坎宫 | 掌底中央 (手腕近处, 月丘附近) | 财禄、子嗣、智谋、潜意识 |
| 艮宫 | 小指根下方 (近手腕处, 月丘) | 田宅、兄弟、晚辈、不动产 |
| 震宫 | 食指根下方 (近木星丘) | 兄弟、出行、惊恐、动力 |
| 巽宫 | 拇指与食指之间虎口处 | 财运、风波、奔波、人际关系 |
| 离宫 | 中指根部 (土星丘上方) | 官禄、名声、显达、智慧光明 |
| 坤宫 | 无名指根部 (太阳丘附近) | 母亲、妻子、田宅、福气 |
| 兑宫 | 小指根部 (水星丘附近) | 子女、口舌、技艺、商贸 |
| 明堂 | 掌心中央 (火星平原) | 一身福禄之所聚, 主财库 |

### 看八卦的要点

- **宫位丰隆**: 该卦肉厚、有气者, 主该事项顺遂。
- **宫位塌陷**: 该卦凹陷、无肉者, 主该事项有亏。
- **宫位有纹冲破**: 主该事项有破败。
- **明堂凹**: 容财, 主有财库; **明堂凸**: 不容财, 主漏。

## 四、主要掌纹 (5大主线)

### 1. 生命线 (Life Line)

- **位置**: 起于食指与拇指之间, 沿金星丘外侧弧形下行至手腕。
- **主管**: 生命力、体质、健康、抗压能力、人生重大事件。
- **看法**:
  - 长而深、弧度大: 体质强, 精力旺。
  - 短: 不必恐慌, 不必然短寿; 仅指生命力起伏的节点。
  - 断裂: 该年份有重大变故 (病、事故、迁徙)。
  - 链状: 体质较弱, 易疲劳。
  - 旁有副生命线 (火星线): 生命力增强, 易危为安。
  - 末端分叉: 晚年迁徙或多变。

**重要警示**: **生命线不预测死亡时间**。生命线是体质与生命力的指示, 而非寿命终点的标尺。古今多有生命线短而长寿者, 反之亦然。

### 2. 智慧线 / 头脑线 (Head Line)

- **位置**: 起于食指与拇指之间 (常与生命线起点相连), 横穿掌心至小鱼际或月丘。
- **主管**: 思考方式、智力、专注力、思维倾向。
- **看法**:
  - 长直: 理性、逻辑强、思维清晰。
  - 短: 思考实际, 不爱钻牛角尖。
  - 下垂至月丘: 想象力丰富、文艺、感性。
  - 上扬: 现实、功利、商业头脑。
  - 与生命线分开 (米字纹形态): 独立、早熟、有主见。
  - 分叉: 多才多艺, 思维有多种倾向。
  - 链状: 注意力难集中。

### 3. 感情线 / 心线 (Heart Line)

- **位置**: 起于小指下方, 横向延伸至食指方向。
- **主管**: 感情、婚姻、心脏健康、对人态度。
- **看法**:
  - 长而清晰、向上扬至食指: 重感情, 婚姻顺。
  - 短: 情感淡漠或专一某人。
  - 链状: 感情复杂、易反复。
  - 断裂: 主感情挫折或心脏问题。
  - 末端分叉: 感情丰富或多角关系。
  - 与智慧线相连 (通贯掌): 性情极端, 爱恨分明。

### 4. 事业线 / 命运线 (Fate Line)

- **位置**: 由手腕向上直行至中指根 (土星丘)。
- **主管**: 事业、职业方向、社会成就。
- **看法**:
  - 清晰深长: 事业稳定有方向。
  - 浅淡或无: 事业多变, 或自由职业。
  - 起点低 (近手腕)、终点高: 早年立志, 一路向上。
  - 起点高: 大器晚成。
  - 中断: 事业有重大转折。
  - 多条事业线: 多元发展, 跨界。
  - **注**: 现代社会"无事业线"不必然差, 自由职业者、艺术家常无明显事业线。

### 5. 婚姻线 (Marriage Line)

- **位置**: 小指根下方, 感情线上方, 短小横纹。
- **主管**: 婚姻、亲密关系、子女。
- **看法**:
  - 一条清晰深长: 婚姻稳定。
  - 多条: 桃花多, 或重要感情关系多。
  - 末端下垂: 婚后多操心 (不必然离婚)。
  - 末端上扬: 配偶有成就。
  - 分叉: 婚姻波折或异地。
  - 岛纹: 婚姻有困境期。

## 六、特殊纹路

### 1. 川字掌

生命线、智慧线、感情线三条并行不相交, 形如"川"字。

- **男**: 主独立、有主见、白手起家。
- **女**: 古谓"克夫", 现代解为"性格强、不依附", 适合独立事业。

### 2. 通贯掌 (断掌, Simian Crease)

智慧线与感情线合为一条横贯手掌的深纹。

- 古谓"男左断掌值千金, 女右断掌守空房"。
- 现代解: 性情极端、专注、固执、爱恨分明; 极适合专业领域深耕。
- 医学角度: 是某些染色体异常的体征 (如唐氏综合征), 但单独出现的断掌大多数为正常变异, 无医学意义。

### 3. 智慧线分叉

- 末端二叉: 双才, 文武或文理兼通。
- 末端三叉: 多才多艺, 思维灵活。

### 4. 特殊符号

| 符号 | 含义 |
|------|------|
| 岛纹 | 出现在某线某段, 主该时期该事项受阻、有困境 |
| 星纹 | 出现在丘上为吉 (荣耀), 出现在线上为凶 (突变) |
| 十字纹 | 多主转折、警示, 在金星丘上为感情大事 |
| 三角纹 | 多主吉, 在掌中为聚财, 在丘上为该方面才华 |
| 方形纹 | 保护纹, 防止灾厄, 出现在生命线断处主"逢凶化吉" |
| 鱼纹 | 大吉, 主财喜双至 (中国独有解读) |
| 流苏纹 (末端散开) | 该线末段精力耗散 |
| 链状纹 | 该段时期反复、不顺 |

## 七、手型分类

### 西方 Cheiro 七型

| 手型 | 特征 | 性格 |
|------|------|------|
| 元素手 (Elementary) | 短粗、僵硬、关节粗 | 体力劳动型, 朴实 |
| 方形手 (Square) | 掌方、指方、关节明显 | 务实、有逻辑、商业型 |
| 圆锥手 (Conic) | 掌圆、指尖渐尖 | 艺术、感性、易冲动 |
| 尖头手 (Pointed/Psychic) | 修长、指尖尖锐 | 灵性、敏感、易脱离现实 |
| 哲学手 (Philosophic) | 长指、关节凸显 | 思辨、深沉、学者型 |
| 实用手 (Useful/Spatulate) | 指尖如铲, 宽厚 | 行动派、创新、运动型 |
| 混合手 (Mixed) | 各型混合 | 多才多艺, 适应性强 (最常见) |

### 中国传统五型 (依五行)

| 手型 | 五行 | 特征 | 性格 |
|------|------|------|------|
| 木形手 | 木 | 修长、骨节凸显, 色青 | 清雅、文气、神经质 |
| 火形手 | 火 | 尖瘦、指尖红 | 热情、急智、性烈 |
| 土形手 | 土 | 厚重、方正、肉厚 | 沉稳、诚信、迟缓 |
| 金形手 | 金 | 方正、色白、骨硬 | 刚毅、果断、原则强 |
| 水形手 | 水 | 圆润、丰满、柔软 | 圆融、智慧、享乐 |

### 看手型的要点

- **形与色须合**: 木形配青色为正; 水形配白胖为正。
- **形与命须配**: 命中喜何五行, 手型属之者助命。
- **男女异**: 男宜阳手 (厚、方、硬), 女宜阴手 (柔、圆、润)。

## 八、指节 / 指节比例

### 三指节意义

每根手指 (拇指除外为两节) 分三节:

- **第一节 (指尖到第一关节)**: 精神、理想、思考。
- **第二节 (中节)**: 理性、推理、现实。
- **第三节 (近掌)**: 物质、欲望、本能。

哪节最长, 哪面最发达。如第一节长者偏理想主义; 第三节肉厚者重物欲。

### 指头与五行 (中国手相)

| 指 | 五行 | 主管 |
|----|------|------|
| 拇指 | 土 | 意志、父母、自我 |
| 食指 | 木 | 抱负、权力、长子 |
| 中指 | 火 | 现实、命运、自我 |
| 无名指 | 金 | 艺术、名声、爱情 |
| 小指 | 水 | 子嗣、口才、商业 |

### 拇指特别看法

- 拇指为"意志指", 反映个人意志强弱与独立性。
- 拇指长 (过食指第二节中点): 主见强、领导才。
- 拇指短: 易受人影响, 依赖性强。
- 拇指能弯曲 (柔软型): 灵活、能屈能伸。
- 拇指僵硬 (难弯): 固执、原则强。

## 九、指甲

指甲不仅看运势, 更是健康的"晴雨表"。

### 形状

| 形状 | 性格 |
|------|------|
| 长方形 | 理性、谨慎 |
| 方形 | 务实、刚直 |
| 圆形 | 圆融、温和 |
| 三角形 (尖) | 敏感、神经质 |
| 扇形 (宽) | 急躁、强势 |

### 健康信号

- **白点**: 古谓"喜事将至", 医学认为是钙锌不足或外伤。
- **横纹 (Beau线)**: 大病后或营养不良的痕迹。
- **竖纹**: 老化、肝肾亏虚。
- **匙状指甲 (反甲)**: 缺铁性贫血常见。
- **杵状指**: 心肺疾病的可能体征 (须就医)。
- **甲色青紫**: 循环不良或寒证。
- **甲色苍白**: 贫血或气血虚。
- **甲面凹凸不平**: 长期慢性病或营养不均。

**警示**: 指甲异常持续存在时, 应优先咨询医生, 而非纯凭手相判断。

## 十、流年法

将主要纹路按时间分段, 推断各年龄段事项。

### 生命线流年 (常用)

以生命线起点 (食指与拇指间) 为0岁, 终点 (手腕) 为100岁, 中间均分。

简化分段法:
- 起点至中指中线垂下点: 约0–30岁。
- 中指中线垂下点至无名指中线垂下点: 约30–50岁。
- 无名指中线垂下点至终点: 约50–100岁。

### 感情线流年

由小指下方为起点 (0岁), 至食指方向为终点。
- 小指至无名指间: 青年 (20–35岁)。
- 无名指至中指间: 中年 (35–50岁)。
- 中指至食指: 老年 (50岁后)。

### 事业线流年

以手腕为起点, 中指根为终点。
- 手腕至智慧线: 0–35岁。
- 智慧线至感情线: 35–50岁。
- 感情线之上: 50岁后。

### 使用提醒

流年法仅为粗略, 实际须结合多线对照、宫位互参。任何时间预测均应留有余地。

## 十一、与现代医学的接口

部分掌纹变化反映身体状况, 但需医学确认:

| 掌纹/手部变化 | 可能的医学意义 |
|--------------|---------------|
| 手掌发红 (鱼际红) | 肝病、酒精性肝硬化 (肝掌) |
| 手掌湿冷 | 自主神经失调、甲状腺机能低下 |
| 手指震颤 | 帕金森、甲亢、低血糖 |
| 通贯掌 (断掌) | 多数为正常变异, 极少与染色体异常相关 |
| 指甲杵状变 | 心肺疾病 |
| 指甲匙状 | 缺铁性贫血 |
| 生命线突然变化 | 重大健康事件后 (但不能预测未发生疾病) |

**重要原则**:
1. 手相可提示"留意健康", 但不能诊断具体疾病。
2. 任何身体异常应优先就医, 而非依赖手相。
3. 反之, "好"的手相也不能替代健康生活方式。
