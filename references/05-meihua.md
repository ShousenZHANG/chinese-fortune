## 一、梅花易数概述

### 1.1 名义来源

"梅花"之名,源自邵雍**观梅占雀**一案:康节先生观梅花,有二雀争枝坠地,因起一卦,断"明日有邻女折花,坠伤其股"——次日果应。后人遂以"梅花"名其法。

### 1.2 心易特征

梅花易数为"心易"——

- **见物起意**:不必摇钱、不必蓍草,见象即起卦。
- **重外应**:占时所闻所见(鸟鸣、风声、人语)皆可为参验。
- **简而活**:法简单,而灵感、体悟之活泼远过六爻。
- **快**:从起卦到判断,数分钟可成。

故梅花易适合:

- 临场断事(突发问题)。
- 答客问(对方随口报数)。
- 见物起意(忽有所感)。
- 长时跨度模糊问题(无法定时辰起六爻者)。

### 1.3 学习次第

1. 熟八卦象数及五行属性。
2. 熟先天卦序(乾1兑2离3震4巽5坎6艮7坤8)。
3. 熟起卦诸法。
4. 熟体用分判与五行生克在断卦中的应用。
5. 学应期判断。
6. 学外应配合。

## 二、起卦方法

### 2.1 总原则

任何起卦法,最终目的:**得上卦、下卦、动爻**。

- 上卦 + 下卦 = 本卦(六爻完整)。
- 动爻位置 = 决定变卦与体用。

八卦序数(先天):**乾1、兑2、离3、震4、巽5、坎6、艮7、坤8**。

### 2.2 年月日时起卦

公式:

- **上卦** = (年支序 + 月数 + 日数) ÷ 8,取余(余0按8)。
- **下卦** = (年支序 + 月数 + 日数 + 时支序) ÷ 8,取余。
- **动爻** = (年支序 + 月数 + 日数 + 时支序) ÷ 6,取余(余0按6)。

地支序数:**子1、丑2、寅3、卯4、辰5、巳6、午7、未8、申9、酉10、戌11、亥12**。

**示例**:乙丑年三月初四申时

- 年支丑=2,月3,日4,时申=9。
- 上卦 = (2+3+4) ÷ 8 = 9 ÷ 8,余1 → 乾。
- 下卦 = (2+3+4+9) ÷ 8 = 18 ÷ 8,余2 → 兑。
- 动爻 = 18 ÷ 6 = 3,余0按6 → 上爻动。
- 本卦:**乾上兑下,天泽履**,上爻动 → 变卦"兑为泽"。

### 2.3 报数起卦

**单数法**(被卜者随报一数):

- 数 ÷ 8 余为上卦。
- 数 + 即时时支序 ÷ 8 余为下卦。
- 总数 ÷ 6 余为动爻。

**双数法**(报两数):

- 第一数 ÷ 8 余 → 上卦。
- 第二数 ÷ 8 余 → 下卦。
- (两数之和) ÷ 6 余 → 动爻。
- (有时加时辰)

**示例**:友人报"37、98"。

- 37 ÷ 8 = 4 余5 → 巽,上卦。
- 98 ÷ 8 = 12 余2 → 兑,下卦。
- (37+98) ÷ 6 = 135 ÷ 6 = 22 余3 → 三爻动。
- 本卦:**巽上兑下,风泽中孚**,三爻动 → 变卦"风雷益"。

### 2.4 声音起卦

听到的声音次数或种类:

- 一声一物为单数 → 按报数单数法。
- 二声、二物为双数 → 按双数法。
- 如**鹊噪七声**——以7÷8余7为艮,加时辰为下卦,合数÷6取动爻。

### 2.5 物象起卦

见物起意——见某物即对应八卦:

- 见龙、马、天象 → 乾。
- 见羊、井泉 → 兑。
- 见火、雉、电、太阳 → 离。
- 见雷、龙、震动之物 → 震。
- 见风、鸡、绳直长 → 巽。
- 见水、雨、豕、月 → 坎。
- 见山、狗、止物 → 艮。
- 见牛、土、布帛 → 坤。

**双物**:上下两物得上下卦,合数 + 时 ÷ 6 = 动爻。
观梅例正属此类:雀(离/兑皆可)+ 梅花(巽,木),得卦后断。

### 2.6 字数起卦

**一字**:观字之笔画或字象。
- 字象法:看字的偏旁象什么卦(如"日"为离、"月"为坎)。
- 笔画法:笔画 ÷ 8 取上下两半,前半为上卦笔数(再 ÷ 8),后半为下卦。

**二字**:第一字笔画 → 上卦;第二字笔画 → 下卦;(笔画总和 + 时支)÷6 → 动爻。

**示例**:友人写"求财"二字问事。

- "求"七画 → 7,÷ 8余7 → 艮,上卦。
- "财"七画 → 7,÷ 8余7 → 艮,下卦。
- 加时辰,如午时午=7,(7+7+7)÷6 = 21÷6 = 3余3 → 三爻动。
- 本卦:**艮为山**,三爻动 → 变卦"山地剥"。

**多字**:总字数为偶,前一半为上卦,后一半为下卦,合数取动爻;总字数为奇,前一字独多一字为上卦。

### 2.7 方位起卦

来人来自某方位,即对应卦:

- 西北来 → 乾;西 → 兑;南 → 离;东 → 震;
- 东南 → 巽;北 → 坎;东北 → 艮;西南 → 坤。

来人方位为上卦;来人衣色或所抱物再得下卦;合时辰 ÷ 6 取动爻。

### 2.8 五行起卦

依被卜者属相五行、或所言事物五行起卦——较少用,多与他法配合。

## 三、本卦 / 互卦 / 变卦

### 3.1 本卦

起卦初得之卦,代表事情**当前状态**。

### 3.2 互卦

取本卦的二、三、四爻为下互,三、四、五爻为上互,组合成新卦。
代表事情**发展过程中的中间状态**或**内在原因**。

例:本卦"风泽中孚"(☴☱)

- 二三四爻:下震卦 ☳
- 三四五爻:上艮卦 ☶
- 互卦:**山雷颐** ☶☳

### 3.3 变卦(之卦)

动爻变其阴阳后得之新卦,代表事情**最终结果**或**未来走向**。

例:本卦"中孚"三爻动 → 第三爻由阴变阳 → 变卦为"风雷益"(☴☳)。

### 3.4 体卦与用卦(关键!)

梅花易数最核心:

- **不动卦 = 体**——代表"我"、"主"、"问事之主体"。
- **动卦 = 用**——代表"事"、"客"、"问事之客体"。

具体判定:

- 若动爻在下三爻 → **下卦为用,上卦为体**。
- 若动爻在上三爻 → **上卦为用,下卦为体**。

记忆:**动爻所在卦为用,不动者为体。**

## 四、体用关系

体卦 vs 用卦的五行生克关系,是梅花易判断的核心。

### 4.1 五种关系

| 关系 | 含义 | 吉凶 |
|------|------|------|
| **比和**(体用同行) | 同心同德 | 大吉 |
| **体克用**(我胜事) | 主动控制事态 | 吉(自胜) |
| **用生体**(事生我) | 事来助我 | 大吉(得益) |
| **体生用**(我泄于事) | 我耗事中 | 小凶(自损、泄) |
| **用克体**(事胜我) | 事来克我 | 大凶(自损) |

口诀:**体克用、用生体——吉;比和——大吉;体生用——泄;用克体——凶。**

### 4.2 旺衰加权

- **体卦宜旺**:得月令、得日辰、得动爻生 → 体强。
- **体卦宜得生**:用卦或互、变卦中有爻生体——大吉。
- **体卦不宜衰**:被月日或动爻克 → 体弱。
- **用卦旺、克体** → 凶上加凶。
- **用卦衰、克体** → 凶小。

### 4.3 互卦、变卦的参与

互卦、变卦中亦有"卦",其五行也参与生克体卦:

- **互卦生体** → 中途有助。
- **互卦克体** → 中途生变。
- **变卦生体** → 结果有益。
- **变卦克体** → 结果受损。

整体看:**体生处皆吉,克处皆凶**。

### 4.4 示例

本卦"火风鼎"(☲☴),五爻动 → 变卦"天风姤"(☰☴)。

- 动爻在上卦 → 上卦为用,下卦为体。
- 体:巽木;用:离火。
- 体生用:木生火——**自泄**,我耗于事。
- 互卦:二三四 = 巽,三四五 = 兑。
  - 巽木同体——比和。
  - 兑金克体——凶。
- 变卦:乾金克体木——**回头克**。

**断**:此事我耗于其中,中途有金气阻(白虎、官非、刀刃),终被金克——凶。当避之。

## 六、应期判断

"应期"——事情何时发生。

### 6.1 卦气定时

按卦本身的五行性质对应季节、月令:

| 卦 | 季节 / 月 | 时辰 |
|----|----------|------|
| 乾 | 戌亥(秋冬之交) | 戌时(19-21) |
| 兑 | 酉(秋) | 酉时(17-19) |
| 离 | 午(夏) | 午时(11-13) |
| 震 | 卯(春) | 卯时(5-7) |
| 巽 | 辰巳(春末夏初) | 辰巳时(7-11) |
| 坎 | 子(冬) | 子时(23-1) |
| 艮 | 丑寅(冬末春初) | 丑寅时(1-5) |
| 坤 | 未申(夏末秋初) | 未申时(13-17) |

### 6.2 用爻地支

变爻地支或体克用爻所对应之地支日,为应期。

### 6.3 三合 / 六合 / 冲日

事成之日:体卦五行对应支之合日。
事破之日:对应冲日。

如体艮(土),
- 合日:辰戌丑未月日(土合);或巳合申(借金土相生)。
- 冲日:卯日(卯戌合化火生土,但单冲艮可视为木冲土)。

应期判定多靠经验,**重要的是看"动爻"与"变卦"的地支**。

### 6.4 应期速断口诀(《梅花》原书)

> 卦遇旺相,其应必近;卦遇休囚,其应必远。
> 动则速,静则缓;急则在日时,缓则在年月。

## 七、外应

"外应"——占卦时所闻所见之**意外征兆**,与卦象相参,可增其验。

### 7.1 外应种类

1. **天象**:风、雨、雷、云、日月光暗。
2. **地象**:草木、土石、声响。
3. **动物**:鸟兽虫鱼之鸣、飞、来去。
4. **人事**:人来人往、所闻言语、所见衣色物色。
5. **器物**:器之破损、合开、有无。

### 7.2 配合解读

- 占时听**喜鹊**叫——青龙、喜信,加吉。
- 占时见**乌鸦**——白虎、凶报,加凶。
- 占时**风雨大作**——震、坎,主惊险。
- 占时**有人持刀过**——金、白虎,主伤、官非。
- 占时**遇人言笑**——朱雀、青龙,事和。

### 7.3 经典准则

> **占卦不外乎卦,占吉凶不外乎应。**

外应非主,辅证而已。卦象本旨为根,外应作枝叶,过分依赖外应则失本。

## 八、各类事项断法

梅花易数因简洁,常用"体用 + 卦象"通用断法。以下为类目要点:

### 8.1 占天时

- 看体卦五行,对应天气:
  - 坎水 → 雨。
  - 离火 → 晴。
  - 震木 → 雷。
  - 巽木 → 风。
  - 艮土 → 阴沉。
  - 坤土 → 阴。
  - 乾金 → 晴明。
  - 兑金 → 阴雨(秋兑泽湿)。
- 用卦生体 → 天助。
- 用卦克体 → 天违所愿。

### 8.2 占地理

- 看上下卦象,组合判方位、地形:
  - 山——艮。
  - 水——坎。
  - 林木——震巽。
  - 平地——坤。
  - 庙宇——离(火,神位)或乾(天)。
- 体旺 → 此地利。
- 体衰 → 不利。

### 8.3 占人物

- 体卦为我,用卦为人。
- 用生体——其人助我;用克体——其人害我。
- 卦象兼看:
  - 乾——尊长、官人、老人。
  - 兑——少女、巫者、口才之人。
  - 离——文人、丽人、武人(火兵)。
  - 震——长男、武人、行动者。
  - 巽——长女、僧道、文士。
  - 坎——中男、水人、智者。
  - 艮——少男、童子、止者。
  - 坤——母、众人、农夫。

### 8.4 占身体

- 体卦五行配脏:
  - 乾兑金——肺、大肠、皮毛。
  - 离火——心、小肠、目。
  - 震巽木——肝胆、神经。
  - 坎水——肾、膀胱、耳。
  - 艮坤土——脾胃、肌肉。
- 体旺无克——病轻或无。
- 体被用克——病重。
- 互、变克体——病转、有反复。

### 8.5 占求名

- 体生用、用克体——名难成。
- 体克用、用生体、比和——名易成。
- 看用卦旺衰:用旺则名实相应。

### 8.6 占求财

- 体克用——得财。
- 用生体——意外之财。
- 体生用——破财、付出。
- 用克体——损财。
- 比和——同好分财。
- 财之大小:看用卦旺相程度。

### 8.7 占婚姻

- 体为己,用为对方。
- 用生体、比和——和合;
- 体克用——己强彼弱,或难合;
- 用克体——彼强己弱,或彼难;
- 体生用——我倾心于彼,彼受益。
- 看互卦——中间是否有阻。

### 8.8 占出行

- 体宜旺,行则顺。
- 体克用——出行顺,得利。
- 用克体——途中有阻、险。
- 卦中有坎——水道险。
- 卦中有震——动而急。
- 卦中有艮——止、慢。

### 8.9 占疾病

- 用卦为病。
- 用克体——病重。
- 体克用——病退。
- 用生体(怪)——病养于身;
- 体生用——病出于外(可解)。
- 互、变看病程:互克体,中段难;变克体,病转重或反复。

### 8.10 占官讼

- 体为己,用为讼对方。
- 体克用——己胜。
- 用克体——己负。
- 比和——和解。
- 用生体——对方让利。
- 体生用——己付罚或解事。

## 九、经典案例

### 9.1 观梅占(《梅花易数·观梅章》)

> 辰年十二月十七日申时,先生偶观梅,见二雀争枝坠地。
> 先生曰:"不动不占,不因事不占,今二雀争枝坠地,怪也。"因占之。

起卦:

- 年支辰=5,月12,日17 → (5+12+17)÷8 = 34÷8 = 4余2 → 兑(2),上卦。
- 加时支申=9 → (5+12+17+9)÷8 = 43÷8 = 5余3 → 离(3),下卦。
- 动爻:43÷6 = 7余1 → 初爻动。
- 本卦:**泽火革**(兑上离下),初爻动 → 变卦"泽山咸"(兑上艮下)。
- 互卦:二三四爻为巽,三四五爻为乾 → 互卦"天风姤"。

断:

- 动爻在下卦 → 离为用,兑为体。
- 兑金(体)受离火(用)克——用克体,**凶**。
- 互卦:巽(克体方木)、乾(生体方金)——参半。
- 变卦:艮土——生兑金体——吉,但变在事后。
- 卦中"少女"(兑)、"明丽"(离)——属伤女子之事。
- 巽为股、为木——折枝伤股。

**先生断曰**:"少女当伤其股。明日,有邻女折花,堕地,伤其股,有损,无大害。"

次日果验。

### 9.2 老人有忧色占

> 邵子(邵雍)某日见一老人面有忧色,问之不答。因起卦——以见老人时辰、方位起卦。

(原文略,要点:)
- 老人——乾。
- 来自西方——兑。
- 卦得"天泽履"。
- 体用判断后,断老人"五日内有食鱼之厄"。
- 后果有鱼骨卡喉之事。

### 9.3 折枝占

> 邵子与友人共行,见有人折椿木一枝。
> 因起卦:椿木为巽(上卦),折枝为震(动象,下卦)。
> (具体起法略——以见物时辰加成。)

断:**老妇明日忌动土;若动则灾**。
后果验。

(此类案例,以见象、闻声、即兴起卦,体现"心易"妙用。)

## 十、与六爻对比

| 维度 | 梅花易数 | 六爻 |
|------|----------|------|
| 起卦 | 多元:时间、报数、字、声、物、方位 | 三铜钱六摇 |
| 工具 | 任意,可无工具 | 必须铜钱 / 卦筒 |
| 装卦 | 极简(本卦+互卦+变卦+体用) | 极繁(纳甲、六亲、六神、世应) |
| 断卦核心 | 体用五行生克 | 用神旺衰、生克、动静、空破墓刑 |
| 强项 | 直观、快捷、即兴 | 精细、可定应期到日 |
| 弱项 | 应期粗略、依赖经验 | 学习成本高、装卦复杂 |
| 适用场景 | 突发问题、即兴答客、模糊大势 | 具体事项、求财、官司、详细分析 |

**何时用梅花**:
- 无法摇钱(车上、对话中)。
- 朋友随口问。
- 直觉来时(见物起意)。
- 大势判断、方向选择。

**何时用六爻**:
- 重要决策(投资、婚姻、官司)。
- 需要定应期、定日辰。
- 想要精细的人事象。
- 长期跟踪事件发展。

二法可相辅相成。古人云:**"易无定法,贵在心通。"**

## 十二、学习建议

1. **先熟八卦**:八卦序数、卦象、五行、性情、对应物象,必须烂熟。
2. **先练起卦**:任何场合,见物即起、心动即起,日练数次。
3. **多读案例**:《梅花易数》原书案例反复推演。
4. **重视外应**:观察占时环境,养成对外应敏感的直觉。
5. **结合六爻**:作为复杂事件的辅助。
6. **戒滥用**:**不动不占,不疑不占**——事不到、心不诚,占之不应。

邵子云:**"易者,逆数也。心定理明,数应于卦,卦应于事——此心易之极也。"**
