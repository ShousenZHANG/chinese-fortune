# 扩展术数索引

本页用于覆盖主流参考文档之外的中国术数、民俗占法和跨流派请求。使用原则：

1. 先确认用户点名的方法，再选择对应小节。
2. 能排盘的用脚本；不能可靠排盘的只做文化说明、输入需求、符号解读，不假装完整推演。
3. 涉及健康、灾祸、死亡、投资、诉讼等高风险主题，遵守 `20-disclaimer.md`。
4. 遇到流派差异，明确说“某派如此取法”，不要把一家口诀说成唯一标准。

## 小六壬

小六壬是民间快占法，常用六宫循环：

1. 大安：稳定、守成、慢中有成。
2. 留连：拖延、反复、信息不畅。
3. 速喜：快讯、喜信、短期有利。
4. 赤口：口舌、争执、误会。
5. 小吉：小成、有人助、可推进。
6. 空亡：落空、虚耗、暂缓。

常见起法：以农历月起宫，农历日起宫，再以时辰落宫。脚本 `scripts/xiaoliuren_cast.py` 支持公历日期时间，也支持直接输入农历月日和时辰。解读时先看落宫，再结合问题类型，不要把快占扩大成长期命运判断。

## 太乙神数

太乙属于“三式”之一，传统上偏国家大事、兵灾、天象、年运。它以太乙局、主客、十六宫、计神、始击、文昌等结构推演。个人日常咨询中，优先说明方法定位；如果用户没有完整局盘，不要编盘。

## 铁板神数

铁板神数以生辰、亲属排行、父母信息等校验条文，核心是“定刻”和条文对应。公开流传版本差异极大，且条文库常有版权或来源问题。没有可靠表库时，只能解释流程和需要的信息，不能凭空生成“铁板条文”。

## 称骨算命

称骨把农历年、月、日、时换成骨重，两、钱相加后查诗诀。它是民俗命理，不宜当作精密八字。若用户只想娱乐，可说明版本差异并给出粗略流程；若要严谨输出，先确认采用哪一版称骨表。

## 七政四余 / 果老星宗

七政四余接近传统星命学，需计算日、月、金木水火土、罗喉、计都、月孛、紫气等星曜在十二宫的位置。它依赖出生地点、精确时间和星历，不适合纯口算。没有星历脚本时，应只解释星曜和宫位含义，或要求用户提供盘。

## 灵签与杯筊

灵签的关键是“签号 + 签诗 + 问题”。不同庙宇签诗不同，不能混用。若用户只要求抽签，可抽签号并说明需要对应庙宇签文；若用户提供签文，可按“签题、典故、诗意、问事、行动建议”解释。

杯筊常见结果：

- 圣杯：一正一反，表示可行或被允许。
- 笑杯：两正，表示问题不清、神明笑而不答。
- 阴杯：两反，表示不允或时机不合。

杯筊适合确认式问题，不适合开放式复杂问题。

## 风水扩展

玄空、八宅、斗首等风水择日必须收集坐向、入住时间、户型、外局形势、使用者年命。没有这些，不要做具体“凶宅/旺财”断语。可先给通用原则，再列出需要补充的信息。

## 输出格式

扩展术数回答建议结构：

1. 说明采用的方法和它适合回答的问题。
2. 列出已知输入和缺失输入。
3. 若可排盘，给出盘式或落宫；若不可排盘，只解释原则。
4. 给出低风险、可执行建议。
5. 加一句边界说明，避免确定性灾祸或高风险建议。
