# 黄历 / 择日 (Chinese Almanac & Auspicious Date Selection)

## 一、黄历概述

黄历, 又称皇历、通书 (粤语作"通胜", 因"书"与"输"同音, 取吉避之)、农历、老黄历, 是中国传统综合性日历。它不仅记录日期, 更将干支、五行、二十八宿、十二建除、宜忌事项、吉时凶时、神煞方位等综合编排, 为日常择吉避凶提供参考。

### 名称由来

- **"黄"**: 传说起源于黄帝时代, 故称"黄历"。又因古代历法由皇室钦定颁布, 民间亦称"皇历"。
- **"通书"**: 取"通晓万事"之意。
- **"通胜"**: 粤语避"输"字讳, 改"书"为"胜"。

### 历代修订

- **汉代**: 太初历首次系统化中国历法。
- **唐代**: 《麟德历》《大衍历》, 已包含择日内容。
- **宋元**: 《纪元历》《授时历》, 择日体系成熟。
- **明清**: 《大统历》《时宪历》, 集大成。
- **清乾隆**: 编纂《钦定协纪辨方书》, 为现代黄历理论基础。
- **现代**: 港澳台至今出版《通胜》, 民间仍广泛使用。

### 黄历的功能

1. **日常择吉**: 嫁娶、入宅、动土、开市等。
2. **指导农事**: 二十四节气、物候。
3. **宗教祭祀**: 祭祀、扫墓、还愿之日。
4. **健康养生**: 不同日子的养生宜忌。
5. **文化记忆**: 传统节日、纪念日。

## 二、黄历每日组成

一页黄历通常包含以下要素:

### 1. 公历日期

公元年月日 (Gregorian Calendar)。

### 2. 农历日期

农历年月日 (Lunar Calendar), 如 "甲辰年三月廿五"。

### 3. 干支

- **年干支**: 如"甲辰"。
- **月干支**: 如"戊辰"。
- **日干支**: 如"丙午"。
- 时干支按日干起遁。

### 4. 五行纳音

干支组合对应六十甲子纳音, 如 "甲子乙丑海中金"。

### 5. 生肖冲

每日干支对应一生肖, 冲对面生肖, 如"丙午日冲鼠"。属冲之人当日宜静不宜动。

### 6. 值神 (十二建除)

每日有一值神: 建、除、满、平、定、执、破、危、成、收、开、闭。详见后文。

### 7. 二十八宿

每日值一宿, 二十八宿循环值日。详见后文。

### 8. 宜 (适宜事项)

当日适宜进行的事项, 如"祭祀、开光、嫁娶、出行、入宅、移徙、安床"。

### 9. 忌 (忌讳事项)

当日不宜进行的事项, 如"开市、动土、安葬、破土"。

### 10. 吉时 / 凶时

每日十二时辰各分吉凶, 用于选具体时刻。

### 11. 吉神宜趋

当日值班之吉神, 如:

| 吉神 | 主何事 |
|------|--------|
| 天德 | 天降福德, 百事皆吉 |
| 月德 | 月之德神, 主吉庆 |
| 月恩 | 上恩下泽, 利贵人 |
| 四相 | 四时旺相, 利建造 |
| 时德 | 时之德, 利办大事 |
| 益后 | 利后代, 嫁娶宜 |
| 续世 | 利子嗣绵延 |
| 天医 | 利求医、服药 |
| 三合 | 三合之日, 万事顺 |
| 六合 | 阴阳和合, 嫁娶吉 |
| 天喜 | 利喜事 |
| 母仓 | 利养育、嫁娶 |
| 五富 | 主财气 |
| 不将 | 利嫁娶 (无嫁娶冲煞) |

### 12. 凶煞宜忌

当日值班之凶煞, 如:

| 凶煞 | 主何忌 |
|------|--------|
| 月破 | 月之冲煞, 万事不宜 (最凶之一) |
| 大耗 | 主破财、损耗 |
| 灾煞 | 主灾祸 |
| 天火 | 主火灾、火气 |
| 月厌 | 主烦扰、不快 |
| 地火 | 主地下事故 |
| 月刑 | 主刑伤 |
| 月害 | 主害人害事 |
| 五虚 | 主空虚损失 |
| 八专 | 主重复事 |
| 触水龙 | 忌行船 |
| 受死 | 大凶, 万事不宜 |
| 重日 | 忌葬, 主重丧 |
| 复日 | 忌葬, 主复丧 |
| 四绝 | 立春前一日, 万事不宜 |
| 四离 | 立秋前一日, 万事不宜 |

### 13. 胎神方位

孕妇当日胎神所在方位, 此方位不可动土、移床、钉钉等, 否则犯胎神, 主孕妇胎儿不安。胎神逐日轮换, 如"碓磨厕外北方"。

| 日支 | 忌 |

| 子 | 子不问卜, 自惹祸殃 |

| 酉 | 酉不会客, 醉坐颠狂 |

| 亥 | 亥不嫁娶, 不利新郎 |

### 15. 喜神 / 财神 / 福神方位

每日吉神所在方位:

| 神位 | 含义 |
|------|------|
| 喜神 | 当日喜庆之事所朝方位 (嫁娶、订婚朝此方) |
| 财神 | 当日财气所在 (求财、开市朝此方) |
| 福神 | 当日福运所在 |
| 阳贵神 / 阴贵神 | 当日贵人方位 |

依日干推算, 每日不同。

## 三、十二建除详解

十二建除是黄历最核心的择日体系, 每日轮值一神。

| 建除 | 含义 | 宜 | 忌 |
|------|------|-----|-----|
| 建 | 建立、开始 | 上任、入学、求职、出行 | 动土、破土 (太刚不柔) |
| 除 | 除旧、扫除 | 扫除、求医、祭祀、解除契约 | 嫁娶、入宅 |
| 满 | 圆满、收纳 | 嫁娶、开市、入宅、祈福 | 动土、安葬 |
| 平 | 平凡、不进不退 | 平整地、铺路 | 求官、求财 |
| 定 | 稳定、入宅 | 入宅、安床、订婚、立约 | 出行、诉讼 |
| 执 | 执守、守成 | 祭祀、捕猎、立约 | 出行、开市 |
| **破** | **破坏、摧毁** | **拆屋、破土 (有限)** | **百事忌, 大事不宜** |
| 危 | 危险、不宜 | 安床、祭祀 | 出行、登高、行船 |
| 成 | 成就、修造 | 嫁娶、入宅、开市、建造 | 诉讼 |
| 收 | 收成、入库 | 入宅、安葬、纳财、收成 | 出行 |
| 开 | 开放、开市 | 开市、上任、入学、出行 | 动土、安葬 |
| 闭 | 闭塞、葬埋 | 安葬、筑堤、封口 | 嫁娶、出行 |

### 十二建除排列规律

每月以"节令"(交节, 非农历初一) 之后第一天为"建", 其后依次循环。
- 寅月 (立春后): 寅日为建。
- 卯月 (惊蛰后): 卯日为建。
- 余类推。

每月轮值十二建除一周再多, 共约 30天。

### 黑道与黄道

- **黄道日**: 除、危、定、执、成、开 (吉日)
- **黑道日**: 建、满、平、破、收、闭 (凶日)

但"黄道日"内仍有具体宜忌, 不能一概而论。

## 五、常见事项择日要点

### 1. 嫁娶

最重要的择日事项之一, 影响一生。

**宜**:
- 月德、天德、月恩、不将、三合、六合日。
- 成、定、开、满日。
- 房、奎、毕、井、张等吉宿。

**忌**:
- 月破、大耗、平、收、闭日。
- 受死、四离、四绝、灭没日。
- 三娘煞日: 每月初三、初七、十三、十八、廿二、廿七 (民俗, 月老不结绳)。
- 杨公忌日: 每月十三、十一、初九、初七、初五、初三 (递减二, 民俗忌出行嫁娶)。

**男女命属相参看**:
- 不冲新人本命 (如新郎属虎, 不选申日)。
- 不冲新娘本命。
- 避开新人父母本命冲日。
- 男女命合婚后, 选共同的吉日。

### 2. 入宅 / 搬家

**宜**:
- 成、开、定、满日。
- 月德、天德、四相、母仓日。
- 张、奎、毕、井等宿。

**忌**:
- 月破、闭日、受死。
- 不冲屋主本命。
- 火星照命日 (个人煞)。

**搬家时辰**: 入宅多选上午, 取"日初升, 气向上"之意。

### 3. 动土 / 开工

**宜**:
- 成、开、满日。
- 月德、天德日。

**忌**:
- 月破、土忌日、土王用事 (四季最后18日为"四季土王", 不动土)。
- 太岁、三煞、五黄方位不动土。
- 戊己日 (土自旺, 慎动)。

### 4. 开业 / 开市

**宜**:
- 开、满日。
- 月恩、四相、五富、天财日。
- 井、毕、奎等宿。

**忌**:
- 月破、平、收、闭日。
- 受死、四离、四绝。
- 老板本命冲日。

### 5. 安葬

**宜**:
- 成、收日。
- 鸣吠日 (主鸣吠, 葬吉)。
- 奎、毕、张、井等吉宿。

**忌**:
- 重日、复日 (主再丧)。
- 月破、受死。
- 四离、四绝、四穷。
- 死者本命冲日。

### 6. 出行

**宜**:
- 建、开、满日。
- 月德、天德、四相日。

**忌**:
- 破、危日。
- 受死、往亡日。
- 出行人本命冲日。

### 7. 求医

**宜**:
- 天医日 (按月支固定: 寅月在卯, 卯月在辰, 依次)。
- 除日 (除病)。
- 平日。

**忌**:
- 月破、受死。
- 未日 (彭祖百忌: 未不服药)。

### 8. 祭祀

**宜**:
- 天德、月德、天赦日。
- 寅、卯、辰、午、未、戌日多吉。

**忌**:
- 寅日 (彭祖百忌: 寅不祭祀)。
- 月破、受死。

### 9. 出火 / 安香

**宜**:
- 满、成日。
- 天德、月德。

**忌**:
- 火星日 (火忌)。
- 主家本命冲。

### 10. 立碑 / 谢土

**宜**:
- 成、收日。
- 与安葬择日类似。

**忌**:
- 重丧、复丧日。

## 六、三煞 / 五黄

### 三煞

每年三煞方位不可动土、修造, 主灾祸。

| 年支 | 三煞方位 |
|------|---------|
| 申子辰年 | 南方 (巳午未) |
| 寅午戌年 | 北方 (亥子丑) |
| 巳酉丑年 | 东方 (寅卯辰) |
| 亥卯未年 | 西方 (申酉戌) |

### 五黄

风水中之凶煞, 每年五黄星飞到的方位为该年"五黄位", 不可动土、不可常居。需查九宫飞星年盘。

例 2024年五黄位在中宫; 2025年五黄位在西北; 2026年五黄位在西方; 余年依紫白飞星推算。

## 七、太岁方位

### 太岁定义

太岁 (Jupiter 木星) 是值年的天神, 每年所在方位忌动土、忌正对、忌冲犯。

### 当年太岁方位

| 年支 | 太岁方位 |
|------|---------|
| 子年 | 北方 (正北) |
| 丑年 | 东北偏北 |
| 寅年 | 东北偏东 |
| 卯年 | 东方 (正东) |
| 辰年 | 东南偏东 |
| 巳年 | 东南偏南 |
| 午年 | 南方 (正南) |
| 未年 | 西南偏南 |
| 申年 | 西南偏西 |
| 酉年 | 西方 (正西) |
| 戌年 | 西北偏西 |
| 亥年 | 西北偏北 |

### 太岁方位禁忌

- **太岁头上不动土**: 太岁所在方位不开工、不动土、不修造。
- **岁破方**: 太岁对面方位 (如子年岁破在午), 同样忌动土。
- **三煞方**: 与太岁同年, 见上节。

### 犯太岁

- **值太岁**: 本命年与太岁相同 (如属猪人遇亥年)。
- **冲太岁**: 本命与太岁相冲 (如属猪人遇巳年)。
- **刑、害、破太岁**: 与太岁相刑、相害、相破。

犯太岁年, 民俗须拜太岁、戴红、佩戴本命符等。

## 八、公历转农历

### 简易方法

公历转农历需依据当年的具体节气、闰月等因素, 手工计算复杂。建议使用:

1. **官方颁布的万年历表**。
2. **黄历软件 / app** (如墨迹万年历、中华万年历)。
3. **脚本计算**: 例如 Python 的 `lunarcalendar` 包或 `cnlunar` 包。

### 脚本范例

```
# scripts/lunar_convert.py
# 使用 lunarcalendar 包
from lunarcalendar import Converter, Solar

solar = Solar(2026, 5, 15)
lunar = Converter.Solar2Lunar(solar)
print(f"农历: {lunar.year}年{lunar.month}月{lunar.day}日")
```

### 关键节气日期

二十四节气是黄历的骨架, 现代精确到分钟。常用节气:

- **立春**: 公历2月3–5日 (新年开始, 节气年起点)
- **春分**: 公历3月20–22日
- **夏至**: 公历6月20–22日
- **秋分**: 公历9月22–24日
- **冬至**: 公历12月21–23日
- **立秋**: 公历8月7–9日

## 十、注意事项

### 黄历的局限

1. **黄历宜忌是综合性日历, 适用于一般情况**, 不针对个人。
2. **真正择日须结合个人八字**: 一人之吉日, 未必是另一人之吉日。
3. **大事择日 (婚嫁、安葬、开工) 应请专业择日师**, 综合考虑:
   - 当事人八字。
   - 当年太岁。
   - 月支节气。
   - 日支冲合。
   - 时辰具体吉凶。
4. **小事可参考黄历**: 一般出行、办事、签约, 看黄历即可。

### 黄历的正确态度

- **黄历是参考, 不是命令**: 不必因黄历凶日就完全不做事。
- **大事谨慎, 小事从俗**: 重大决策可避凶趋吉; 日常事不必拘泥。
- **现代生活的弹性**: 工作日不易完全避凶日, 可在凶日内选吉时。
- **心理作用**: 信则有, 不信则无; 但避免完全机械化。

### 综合择日的原则

1. **避大凶**: 月破、受死、四离、四绝等大凶日不做大事。
2. **趋大吉**: 月德、天德、三合、六合日做大事。
3. **不冲本命**: 当事人本命不被冲, 否则吉日变凶。
4. **不犯太岁**: 不在太岁方位动土。
5. **节令为本**: 节气交接时刻为重要分水岭。

---

**黄历之用, 在于警觉时序、协调人事, 非命定吉凶。慎择良辰, 谨守本心。**
