# 八字四柱 (BaZi / Four Pillars of Destiny)

> 子平之学,源于唐李虚中,成于宋徐子平。以人生年月日时四柱八字推断命运,合于阴阳五行之理。

---

## 一、八字概述

### 1.1 什么是八字

八字,又称"四柱八字"或"子平命理",是中国传统命理学中最重要的预测体系之一。它以人出生的**年、月、日、时**四个时间维度为基础,每个维度由一个天干和一个地支组合而成,共得四柱八个字,故称"八字"。

> 《滴天髓》云:"欲识三元万法宗,先观帝载与神功。"
> 白话:要明白天干地支阴阳五行的奥妙,先要观察天地造化之功。

### 1.2 为何取四柱

四柱对应宇宙时空四个维度:

| 柱 | 时间单位 | 主管时段 | 代表人物 |
|---|---|---|---|
| 年柱 | 60甲子(60年) | 1—15岁(童限) | 祖辈、根源、社会大环境 |
| 月柱 | 节气(月) | 16—30岁(少年) | 父母、兄弟、青年事业 |
| 日柱 | 日干为命主 | 31—45岁(中年) | 自身、配偶、中年成就 |
| 时柱 | 时辰(2小时) | 46岁后(晚年) | 子女、晚年、归宿 |

- **日干**为"命主"(我),是全盘的核心,所有十神都以日干为参照定位。
- **日支**为"夫妻宫",代表配偶宫位。

## 二、排盘步骤

### 2.1 年柱:以立春为界

> **核心原则**:八字年柱的起讫以**立春**为界,而非农历正月初一。

立春一般在公历2月3—5日,具体时刻精确到分。立春前出生,年柱仍属上一年;立春后出生,年柱属当年。

**例**:1990年农历正月初一(公历1990年1月27日)出生者,因尚未交立春(1990立春为2月4日11时14分),年柱仍为**己巳**(1989年),而非庚午(1990年)。

**年柱速查**:由公历年份推算
- 公元年份个位数 → 天干(年份末位对应表):
  - 4=甲、5=乙、6=丙、7=丁、8=戊、9=己、0=庚、1=辛、2=壬、3=癸
- 公元年份除以12取余 → 地支:
  - 余4=子、5=丑、6=寅、7=卯、8=辰、9=巳、10=午、11=未、0=申、1=酉、2=戌、3=亥

**例**:2024年 → 末位4=甲、2024÷12余8=辰 → **甲辰年**。

### 2.3 日柱:六十甲子轮转

日柱不可口诀推算,需查万年历或以已知日柱为基点用60甲子循环推算。

**六十甲子表**:
```
01甲子 02乙丑 03丙寅 04丁卯 05戊辰 06己巳 07庚午 08辛未 09壬申 10癸酉
11甲戌 12乙亥 13丙子 14丁丑 15戊寅 16己卯 17庚辰 18辛巳 19壬午 20癸未
21甲申 22乙酉 23丙戌 24丁亥 25戊子 26己丑 27庚寅 28辛卯 29壬辰 30癸巳
31甲午 32乙未 33丙申 34丁酉 35戊戌 36己亥 37庚子 38辛丑 39壬寅 40癸卯
41甲辰 42乙巳 43丙午 44丁未 45戊申 46己酉 47庚戌 48辛亥 49壬子 50癸丑
51甲寅 52乙卯 53丙辰 54丁巳 55戊午 56己未 57庚申 58辛酉 59壬戌 60癸亥
```

**日柱公式**(基准法):以1900年1月1日为癸亥日(序号60)。计算公式较复杂,实务中应由命理程序处理,或使用《万年历》查表。简化算法:

```
设 y = 公历年份
设 m = 月份(若 m ≤ 2, 则 y = y - 1, m = m + 12)
设 d = 日
C = y / 100(整数)
y = y % 100
i = (4C + C/4 + 5y + y/4 + 3(m+1)/5 + d - 3) % 60
若 i ≤ 0, i += 60
i 即为六十甲子序号。
```

**注意**:闰年、儒略历转格里历需修正。生产代码请用现成命理库。

### 2.5 真太阳时校正

八字所用为**真太阳时**(当地太阳过中天为正午),非北京时间(东经120°标准时)。

**经度修正公式**:
```
真太阳时 = 北京时间 ± (出生地经度 - 120) × 4 分钟
```

经度大于120°(东部)加,小于120°(西部)减。

**例**:乌鲁木齐(东经87.5°)某人北京时间正午12:00出生,
真太阳时 = 12:00 - (120 - 87.5) × 4 分钟 = 12:00 - 130分钟 = 09:50
应取**巳时**而非午时。

进一步精确需加"均时差"(地球椭圆轨道及黄赤交角导致的时差,±16分钟),严谨命理软件会处理。

## 三、五行旺衰判定

> 《滴天髓》云:"得时俱为旺论,失时便作衰看。"
> 白话:得时令则旺,失时令则衰。

判断日主(日干)旺衰是八字分析的核心步骤,直接决定用神选取。

### 3.1 月令为先(最重)

月支是出生的"季节",对全局五行旺衰影响最大,通常占判定权重的40—50%。

**月令对日主的关系**:
- 月支生日主 → 印
- 月支同日主 → 比劫
- 月支为日主所克 → 财
- 月支克日主 → 官杀
- 月支被日主所生 → 食伤

判断**当令**:
- 春月(寅卯辰):木旺,火相,水休,金囚,土死
- 夏月(巳午未):火旺,土相,木休,水囚,金死
- 秋月(申酉戌):金旺,水相,土休,火囚,木死
- 冬月(亥子丑):水旺,木相,金休,土囚,火死
- 四季月(辰未戌丑):土旺

### 3.2 通根

**通根**:天干在地支中找到本气或余气,谓之"通根"。

| 天干 | 通根地支 | 强度 |
|---|---|---|
| 甲乙 | 寅(本气)、卯(本气)、亥(余气)、未辰(余气) | 寅最强 |
| 丙丁 | 巳(本气)、午(本气)、寅(余气)、戌未(余气) | 午最强 |
| 戊己 | 辰戌丑未(本气)、巳午(余气) | 四库最强 |
| 庚辛 | 申(本气)、酉(本气)、巳(余气)、戌丑(余气) | 酉最强 |
| 壬癸 | 亥(本气)、子(本气)、申(余气)、辰丑(余气) | 子最强 |

> **通根口诀**:"得一比肩不如得一余气,得二余气不如得一中气,得三中气不如得一本气。"

### 3.3 党众(帮扶)

天干透出的比劫和印星(同气与生气)即为"党众",是日主的助力。

- **比劫**(同我):甲见甲、乙、寅、卯
- **印星**(生我):甲见壬、癸、亥、子

党众数量越多,日主越强。

### 3.4 旺相休囚死

**五行四时旺相**:

| 五行/季节 | 春(寅卯辰) | 夏(巳午未) | 秋(申酉戌) | 冬(亥子丑) | 四季(辰未戌丑) |
|---|---|---|---|---|---|
| 木 | 旺 | 休 | 死 | 相 | 囚 |
| 火 | 相 | 旺 | 囚 | 死 | 休 |
| 土 | 死 | 相 | 休 | 囚 | 旺 |
| 金 | 囚 | 死 | 旺 | 休 | 相 |
| 水 | 休 | 囚 | 相 | 旺 | 死 |

- **旺**:当令(同气得时)
- **相**:被生(我所生者来扶我)
- **休**:生他(生人者退气)
- **囚**:克他(我克者反耗)
- **死**:被克(克我者最弱)

### 3.5 综合判定流程

1. 看**月令**:日主当令(旺/相)→ 基础强;失令(休/囚/死)→ 基础弱
2. 看**通根**:日干在四支有几个根?根越多越旺
3. 看**党众**:四柱透出多少比劫、印星?
4. 看**克泄耗**:克我(官杀)、泄我(食伤)、耗我(财)的数量与力量
5. 综合衡量:**身旺 / 身弱 / 中和**

> 一般规则:
> - 月令得令 + 多通根 + 党众多 → 身旺
> - 月令失令 + 无根少党 + 克泄耗多 → 身弱
> - 介于二者之间 → 中和(从中和论命最和谐)

## 四、十神详解

十神是以日干为基准,对其他七字(年干、月干、时干、年支、月支、日支、时支藏干)的五行关系进行命名。

### 4.1 十神定义表

| 十神 | 关系 | 阴阳 | 简称 |
|---|---|---|---|
| 比肩 | 同我 | 同性(同阴或同阳) | 比 |
| 劫财 | 同我 | 异性 | 劫 |
| 食神 | 我生 | 同性 | 食 |
| 伤官 | 我生 | 异性 | 伤 |
| 偏财 | 我克 | 同性 | 才 |
| 正财 | 我克 | 异性 | 财 |
| 七杀 | 克我 | 同性 | 杀(偏官) |
| 正官 | 克我 | 异性 | 官 |
| 偏印 | 生我 | 同性 | 枭(枭神) |
| 正印 | 生我 | 异性 | 印 |

**以日干甲木为例**:
- 比肩=甲;劫财=乙
- 食神=丙;伤官=丁
- 偏财=戊;正财=己
- 七杀=庚;正官=辛
- 偏印=壬;正印=癸

### 4.2 十神详解

#### 比肩(比)
- **义**:同我者,如兄弟、朋友、同事
- **吉**:身弱时为帮扶,主合作、独立、自尊、毅力
- **凶**:身旺时为忌神,主争夺、破财、克妻、口舌
- **宫位**:年=兄长;月=兄弟姐妹;日支=配偶宫(身旺则克妻);时=朋友、同辈
- **职业**:合伙、运动员、独立工作者

#### 劫财(劫)
- **义**:异性的兄弟,夺财之神
- **吉**:身弱有比劫为根,助身担财官
- **凶**:破财、克妻、口舌、暴躁;阳刃为劫财之极端
- **特征**:外向、好斗、敢于冒险
- **职业**:武职、销售、竞技

#### 食神(食)
- **义**:我所生之同性,温和的表达与享受
- **吉**:衣食无忧、福寿、子女(女命)、聪明、艺术
- **凶**:过多则懒散、心高志大而无成
- **宫位**:女命食神为子女、男命为下属、徒弟
- **职业**:餐饮、教师、艺术、设计、技术

> 《子平真诠》云:"食神为吉,与七杀两不相伤,真食神也。"

#### 伤官(伤)
- **义**:我所生之异性,聪明而锐利
- **吉**:才华横溢、艺术天分、口才、表演力
- **凶**:伤官见官为祸百端;克夫(女命);好强、傲慢、惹是非
- **配置**:
  - **伤官佩印**:才情兼备,大富大贵
  - **伤官生财**:才华转化为财富
  - **伤官见官**:大忌(尤其女命克夫)
  - **伤官伤尽**:无官星反成大格
- **职业**:艺术、媒体、律师、口才类工作

> 《滴天髓》云:"伤官见官,为祸百端,然有可见者,有不可见者。"

#### 正财(财)
- **义**:我所克之异性,正当之财
- **吉**:勤俭、正当收入、妻子(男命)、信用
- **凶**:贪财坏印、为财所累
- **宫位**:日支正财=贤妻、月正财=父亲、时正财=晚年财
- **职业**:稳定职业、会计、金融、实业

#### 偏财(才)

## 六、格局判定

## 七、大运排法

> 大运是先天命局之外的"后天运程",每柱主管10年,与本命合参。

### 7.1 阴阳年与男女顺逆

**阳年**:甲、丙、戊、庚、壬年(年干为阳干)
**阴年**:乙、丁、己、辛、癸年(年干为阴干)

| 性别 | 阳年 | 阴年 |
|---|---|---|
| 男命 | 顺排 | 逆排 |
| 女命 | 逆排 | 顺排 |

**起点**:从月柱开始,顺/逆排60甲子。

**例**:某男命,1990年(庚午,阳年)甲申月生
- 男命阳年→顺排
- 月柱甲申之后顺排:乙酉、丙戌、丁亥、戊子、己丑、庚寅...
- 大运依次为乙酉、丙戌、丁亥...

### 7.2 起运岁数

**起运公式**:从出生日起算到下一个"节"(顺排)或上一个"节"(逆排)的天数,除以3,得起运岁数。

```
起运岁 = 距节气天数 ÷ 3
余1天 = 加4个月
余2天 = 加8个月
```

> 古法:三日折一岁,一日折四月,一时折十日(更精确)。

**例**:出生于公历1990年8月20日,顺排,下一节为"白露"(9月8日)。
- 距节天数 = 9月8日 - 8月20日 = 19天
- 起运岁 = 19 ÷ 3 = 6余1
- 起运 = 6岁 + 4个月 = 6岁4个月

故此命大运:6—16岁乙酉、16—26岁丙戌、26—36岁丁亥...

### 7.3 大运的看法

每柱大运10年,前5年重天干,后5年重地支。结合本命:
- 大运十神是用神 → 大吉之运
- 大运十神是忌神 → 不利之运
- 大运与命局合冲刑害 → 重要变化

> 《滴天髓》云:"何知其人富,财气通门户;何知其人贵,官星有理会。"

## 八、流年解读

流年即每年的天干地支(从立春到下年立春),与本命及大运合参:

1. **本命视角**:流年十神为何?是用神还是忌神?
2. **大运视角**:流年与大运是否相合相冲?
3. **岁运并临**:大运与流年干支相同,为特殊提示(吉凶皆放大)
4. **天克地冲**:流年天干克日干、地支冲日支(冲提纲),易有大事
5. **太岁**:流年地支即太岁,犯太岁(刑冲合害)需谨慎

**流年六冲常见后果**:
- 子午冲:心血管、烦躁、迁移
- 丑未冲:亲情纠纷、田宅变动
- 寅申冲:出行、外伤、变动
- 卯酉冲:口舌、感情、肝胆
- 辰戌冲:罡煞冲、官非、变动
- 巳亥冲:奔波、文书、贵人变化

## 九、用神 / 喜神 / 忌神

### 9.1 用神选取四法

**用神**是平衡命局的关键之神。古法有四:

1. **扶抑用神**:身旺扶其所克所泄(财、官、食伤);身弱扶其同党(比劫、印)
2. **病药用神**:命局有"病"(过旺或受克之神),取制此病之神为药
3. **调候用神**:寒暖燥湿不调,以五行调候为用
   - 冬生水旺寒木,以丙火调候
   - 夏生火炎土燥,以壬水调候
   - 春生木旺，以庚金为用
   - 秋生金白水冷,以丙丁调候
4. **通关用神**:命局两强相争,取其中介五行通关
   - 木土相战,取火通关(木生火、火生土)
   - 水火相战,取木通关
   - 金木相战,取水通关

### 9.2 喜忌神

- **用神**:最关键的一个五行/十神
- **喜神**:生用神之神(辅助用神者)
- **闲神**:不直接影响命局的五行
- **忌神**:与用神相反之神(克用神者)
- **仇神**:生忌神之神

### 9.3 用神应用

用神选定后:
- **大运流年**逢用神/喜神 → 顺
- **大运流年**逢忌神/仇神 → 逆
- **方位**:用神方位为吉(如用火则南方利)
- **颜色**:用神色为吉(用火则红色)
- **行业**:对应用神五行的行业
- **配偶**:选用神五行/十神突出的对象

> 《滴天髓》云:"用神不可损伤,日主最宜健旺。"

## 十、六亲

### 10.1 父母

- **正印**=母亲,**偏财**=父亲(男女通用)
- **印星受克**(财坏印)→ 母多波折
- **财星受冲**→ 父易出问题
- **月柱**=父母宫,月柱吉神多则得父母之助

### 10.2 兄弟姐妹

- **比肩**=同性兄弟姐妹,**劫财**=异性兄弟姐妹
- **比劫旺**而身弱→兄弟多有助力
- **比劫旺**而身旺→兄弟多反争夺
- **比劫被冲克**→ 兄弟不和或损伤

### 10.3 配偶

- **男命**:正财=妻,偏财=妾;**女命**:正官=夫,七杀=情人/二夫
- **日支**=夫妻宫(无论男女)
- 日支为用神 → 配偶有助
- 日支为忌神 → 配偶不和或多波折
- 日支逢冲 → 婚姻易变
- 日支逢合(尤其六合)→ 婚姻美满
- 日支为桃花、咸池 → 异性缘强(或风流)

**男命特殊**:
- 财星藏支不透 → 妻贤而内敛
- 财星过多过旺 → 男命怕妻、易有外遇
- 比劫多 → 损妻、克妻

**女命特殊**:
- 官杀混杂 → 感情复杂、多婚之兆
- 伤官见官 → 大忌(克夫)
- 食神制杀(无伤官)→ 旺夫
- 日支食神 → 嫁好夫

### 10.4 子女

- **男命**:七杀=儿,正官=女(古法);现代多用食伤
- **女命**:食神=女,伤官=儿(食伤为子女星)
- **时柱**=子女宫
- 时柱吉神 → 子孝
- 时柱凶煞 → 子女有忧
- 时支冲克日支 → 子女顶撞父母
- 子女星受克 → 子女缘薄

## 十二、各领域解读框架

### 12.1 财运

1. **财星位置**:年月支为早年财、日时为中晚年财
2. **财星强弱**:身弱财旺则担不动财(易因财致祸);身旺财弱靠食伤生财
3. **食伤生财**:有食伤生财 → 持续之财、靠才能赚钱
4. **官护财**:有官星护财 → 财不易被劫
5. **财库**:辰戌丑未中藏财(如甲日见辰中乙=财库);冲开财库为发财之兆
6. **格局**:正财格稳富,偏财格暴富易破,食伤生财格才富

## 十四、常见误区

### 14.1 排盘误区

1. **以农历正月初一为年柱起点** → 错;应以**立春**为界
2. **以农历每月初一为月柱起点** → 错;应以**节(非气)**为界
3. **以子时为日界存争议**:
   - 主流:23:00—00:00为夜子时(算前一日),00:00—01:00为早子时(算当日)
   - 另派:23:00—01:00整体算下一日(子时换日)
   - 实务建议:用前者(夜子时归前日,早子时归后日)
4. **忽略真太阳时校正** → 偏离时柱判断;西北、远东地区误差极大
5. **忽略时区**:民国前不同地区有地方时;海外应用当地经度

### 14.2 判旺衰误区

1. **只数十神个数不看力量** → 错;月令、通根、透干权重不同
2. **只看天干不看地支** → 错;地支力量更稳定
3. **不看刑冲合化** → 错;合化变质、冲克失力,需综合判断
4. **机械套从格** → 错;从格条件极严,极少见;勉强套从格则全错

### 14.3 取用神误区

1. **不分扶抑、调候、通关、病药混用** → 取用神需对症
2. **忽略调候** → 冬寒夏热燥湿之命,调候用神优于扶抑
3. **以多者为用** → 错;多者常为忌,少者(平衡者)为用
4. **忽略大运** → 用神需考虑大运配合,某些用神在某些大运失效

### 14.4 解读误区

1. **断生死**:不可武断,需多方印证;古法戒"轻言生死"
2. **以一神煞定吉凶**:神煞需配合格局、十神综合判断
3. **男女不分**:男女十神主管不同(男以财为妻、女以官为夫)
4. **绝对论命**:八字提示倾向，非宿命；修身、积德、风水可改命
