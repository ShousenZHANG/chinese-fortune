# 紫微斗数 (Zi Wei Dou Shu)

> 紫微斗数,以星曜布列十二宫,推人之命运。盖以紫微为帝座,统御群星,故名"斗数"。

---

## 一、紫微斗数概述

### 1.1 起源与传承

紫微斗数相传起源于五代末宋初,由著名道家**陈抟**(陈希夷,字图南,号扶摇子)所创,后世亦有归功于**吕纯阳**(吕洞宾)者。其学融合道家阴阳五行、星象观测与命理推算,自成体系。

> 《紫微斗数全书》序云:
> "希夷仰观天上之星,运用人间之命,作为此书。"
> 白话:陈希夷仰观天上的星辰,运用在推断人间命运上,创作了这部书。

紫微斗数主要流派:
- **三合派**(传统派):重视星曜在三方四正中的关系
- **飞星派**(钦天门):重视四化飞星在宫位间的流动
- **中州派**:王亭之等推广,综合应用
- **现代综合派**:三合+飞星合用

### 1.2 与八字之别

| 项目 | 八字 | 紫微斗数 |
|---|---|---|
| 基础 | 干支阴阳五行 | 星曜配宫位 |
| 起算 | 立春换年、节气换月 | 阴历(农历)月日 |
| 时间精度 | 时辰为最小单位 | 时辰为最小单位 |
| 推断方式 | 用神十神格局 | 星曜亮度、宫位组合、四化飞星 |
| 优势 | 趋势宏观、五行调候 | 事项细致、十二宫分明 |
| 劣势 | 事项细节不易精准 | 五行调候、用神不显 |
| 西方占星比较 | 较抽象 | 接近,但更复杂 |

> 古谚:"八字看流年大势,紫微看事项细节。"二者宜合参。

## 二、命盘十二宫

### 2.1 十二宫名称与主管

紫微命盘以**地盘十二地支**(子丑寅卯辰巳午未申酉戌亥)为基础,十二宫按逆时针(或顺时针,流派不同)分布,主管人生十二大领域。

| 宫位 | 主管内容 |
|---|---|
| 命宫 | 命主基本性格、外貌气质、人生主轴、整体命运 |
| 兄弟宫 | 兄弟姐妹关系、平辈相处、合伙人 |
| 夫妻宫 | 配偶相貌、性格、婚姻状况、感情运 |
| 子女宫 | 子女缘分、性格、教育、性能力、创意 |
| 财帛宫 | 财源、理财方式、收入水平、动产 |
| 疾厄宫 | 健康、体质、易患疾病、灾厄 |
| 迁移宫 | 外出运、旅行、移民、外地发展 |
| 仆役宫(交友宫) | 朋友、下属、同事关系 |
| 官禄宫(事业宫) | 事业、职业、地位、上司关系 |
| 田宅宫 | 房产、不动产、家宅、家族 |
| 福德宫 | 享受、嗜好、内心世界、精神状态、福气 |
| 父母宫 | 父母关系、长辈、上司、文书、相貌(隐含) |

### 2.2 身宫(叠宫)

**身宫**不是单独的第十三宫,而是依生时叠放在十二宫中**特定五宫**之一(命、夫妻、财帛、迁移、官禄、福德)。

身宫意义:
- **后天努力的方向**(命为先天,身为后天)
- **35岁后渐显**:命主中年后人生重心
- **与命宫合参**:命宫主先天定性,身宫主后天发展

**身宫安法**:从寅宫起正月,顺数生月,再从该宫顺数生时所至宫位即为身宫(详见排盘步骤)。

身宫所在的六个可能位置:

| 出生时 | 身宫位置(相对命宫) | 含义 |
|---|---|---|
| 子、午 | 身居命宫 | 命身同宫,先天后天合一 |
| 卯、酉 | 身居夫妻宫 | 婚姻为后半生重心 |
| 寅、申 | 身居迁移宫 | 外出发展为主 |
| 巳、亥 | 身居财帛宫 | 财运为重心 |
| 辰、戌 | 身居官禄宫 | 事业为重心 |
| 丑、未 | 身居福德宫 | 享受、精神生活为重 |

## 三、排盘步骤

紫微斗数排盘较八字复杂,涉及**多重起例**与**百余星曜安星**。以下为核心步骤。

### 3.1 准备工作:出生时间换算

- **必须用农历**(阴历)的月、日
- 时辰用真太阳时(同八字)
- 闰月处理:十五日前算上月,十五日后算下月(主流)

### 3.2 第一步:起命宫

**口诀**:寅宫起正月,顺数到生月;再从生月所到之宫,**逆数生时**(子时算1),所止之处即为命宫。

**详细步骤**:
1. 寅宫=正月,卯宫=二月,辰宫=三月... 顺数至生月所在之宫
2. 从生月所在宫起子时,逆数:子=该宫,丑=逆一宫,寅=逆二宫... 至生时

**例**:农历五月寅时生
1. 寅宫起正月→卯二月→辰三月→巳四月→**午五月**
2. 从午宫起子时,逆数:子=午、丑=巳、**寅=辰**
3. **命宫=辰宫**

### 3.3 第二步:起身宫

**口诀**:同命宫起例,但**顺数生时**(子时=生月所在宫)。

**例**(承上):
1. 生月在午宫
2. 子时=午、丑=未、**寅=申**
3. **身宫=申宫**

### 3.4 第三步:安十二宫

命宫确定后,按**逆时针**顺序安其余十一宫:

**顺序**:命宫 → 兄弟宫 → 夫妻宫 → 子女宫 → 财帛宫 → 疾厄宫 → 迁移宫 → 仆役宫 → 官禄宫 → 田宅宫 → 福德宫 → 父母宫

**例**(承上,命宫在辰):
- 命宫=辰、兄弟=卯、夫妻=寅、子女=丑、财帛=子、疾厄=亥
- 迁移=戌、仆役=酉、官禄=申、田宅=未、福德=午、父母=巳

### 3.5 第四步:定五行局

**五行局**由**命宫地支**与**生年天干**配合而定,共五种:

- 水二局
- 木三局
- 金四局
- 土五局
- 火六局

**定局口诀**(纳音五行法):

> 命宫地支与生年天干干支组合后,查其**纳音五行**即为五行局。

**简易表**:

| 命宫地支\年干 | 甲己 | 乙庚 | 丙辛 | 丁壬 | 戊癸 |
|---|---|---|---|---|---|
| 子、丑 | 水二局 | 火六局 | 土五局 | 木三局 | 金四局 |
| 寅、卯 | 火六局 | 土五局 | 木三局 | 金四局 | 水二局 |
| 辰、巳 | 木三局 | 金四局 | 水二局 | 火六局 | 土五局 |
| 午、未 | 土五局 | 木三局 | 金四局 | 水二局 | 火六局 |
| 申、酉 | 金四局 | 水二局 | 火六局 | 土五局 | 木三局 |
| 戌、亥 | 木三局 | 金四局 | 水二局 | 火六局 | 土五局 |

(实际表略有差异,需以原典《紫微斗数全书》对照纳音表为准)

五行局有两大作用:
- 决定**安紫微星**之位置
- 决定**大限起讫岁数**(几岁起运)

### 3.6 第五步:安紫微星

**方法**:根据**五行局**与**农历生日**,查表定紫微所在宫位。

**算法**(口算法):
```
设 D = 农历生日数 (1—30)
设 N = 五行局数 (水2、木3、金4、土5、火6)

商 q = D ÷ N (向下取整)
余 r = D mod N
若 r = 0, 紫微 = 由商 q 数到对应宫
若 r ≠ 0:
    若 (N - r) 为偶数: 紫微 = (商 q + (N-r)/2) 后再倒推 (N-r) 宫
    若 (N - r) 为奇数: 紫微 = ... (复杂,需查表)
```

| 初六 | 辰 | 卯 | 巳 | 未 | 寅 |
| 初七 | 辰 | 午 | 寅 | 子 | 戌 |

### 3.7 第六步:安14主星

紫微星定位后,其他13主星按**两大星系**布列:

**紫微星系(北斗)6星**:紫微、天机、太阳、武曲、天同、廉贞

**布列规则**(以紫微为起点):
- 紫微于本宫,顺一为天机(空),逆二为太阳;太阳逆一为武曲;武曲逆一为天同;天同逆三为廉贞

| 巳 | 辰 | 寅 | 丑 | 子 | 酉 |
| 午 | 巳 | 卯 | 寅 | 丑 | 戌 |
| 未 | 午 | 辰 | 卯 | 寅 | 亥 |

**天府星系(南斗)8星**:天府、太阴、贪狼、巨门、天相、天梁、七杀、破军

**天府位置**:与紫微相对,以**寅—申宫**为基准,与紫微"对宫互旋"。

具体规则:
- 紫微在子,天府在辰;紫微在丑,天府在卯;紫微在寅,天府在寅(同宫);
- 紫微在卯,天府在丑;紫微在辰,天府在子;紫微在巳,天府在亥;
- 紫微在午,天府在戌;紫微在未,天府在酉;紫微在申,天府在申(同宫);
- 紫微在酉,天府在未;紫微在戌,天府在午;紫微在亥,天府在巳;

**天府八星(顺布)**:
- 天府本宫
- 太阴 = 天府顺一宫
- 贪狼 = 天府顺二宫
- 巨门 = 天府顺三宫
- 天相 = 天府顺四宫
- 天梁 = 天府顺五宫
- 七杀 = 天府顺六宫
- 破军 = 天府顺十宫(隔三宫)

**实例**(紫微在午,天府在戌):
- 天府=戌、太阴=亥、贪狼=子、巨门=丑、天相=寅、天梁=卯、七杀=辰、破军=申

### 3.8 第七步:安副星

副星(辅佐曜)分类:

**14辅星**(略,可见后文)

**六吉星**:左辅、右弼、文昌、文曲、天魁、天钺
- **左辅、右弼**:按生月安(辅:辰起正月顺数;弼:戌起正月逆数)
- **文昌、文曲**:按生时安(昌:戌起子时逆;曲:辰起子时顺)
- **天魁、天钺**(天乙、玉堂贵人):按生年干安

**六煞星**:擎羊、陀罗、火星、铃星、地空、地劫
- **擎羊、陀罗**:按生年禄前后(禄前一宫=羊、禄后一宫=陀)
- **火星、铃星**:按生年支与生时安(命局四火不同)
- **地空、地劫**:按生时安(空:亥起子时逆;劫:亥起子时顺)

**四化星**:化禄、化权、化科、化忌(由生年天干决定四颗主星各得一化)

**其他杂曜**(数十种,略):红鸾、天喜、天马、天姚、咸池、孤辰、寡宿、华盖、龙池、凤阁、三台、八座、台辅、封诰、天哭、天虚、天巫、天月、阴煞...

## 四、14主星详解

### 4.1 紫微(帝座/北斗主)

- **五行**:己土(中央)
- **属性**:尊贵、领导、孤独、自负
- **性格**:威严稳重、有领导欲、自尊心强、好面子、有恻隐之心
- **吉曜会照**:左辅右弼相夹 → 大富大贵(君臣庆会);文昌文曲会 → 文武双全
- **凶煞同宫**:擎羊陀罗 → 孤独刚愎;化忌 → 烦闷孤高
- **入十二宫**:
  - 命宫:领导能力强,但易孤高、独断
  - 夫妻:配偶有地位、性强
  - 财帛:稳定大财
  - 官禄:管理职、领导职
- **庙旺**:午(庙)、寅申(旺)
- **落陷**:无落陷,但卯酉为平,需辅佐

> 《紫微斗数全书》:"紫微帝座,化气曰尊,后天斗数之主。"

### 4.2 天机(智慧星/南斗第三主)

- **五行**:乙木
- **属性**:智慧、机变、动静、宗教悟性
- **性格**:聪明灵活、思虑周密、好动、易紧张、神经质
- **吉**:化科 → 学问名声;会文昌 → 才学;天梁会 → 谋略(机月同梁格)
- **凶**:化忌 → 多疑、心神不宁;煞会 → 投机失误
- **入十二宫**:
  - 命宫:聪明机变、易思虑过度
  - 夫妻:配偶聪明但善变,年龄差距大较好
  - 兄弟:兄弟少而精
  - 迁移:在外多奔波,适合动中求财
- **庙旺**:卯(庙)、巳午(旺)
- **落陷**:丑未(陷)

### 4.3 太阳(中天主星)

- **五行**:丙火
- **属性**:光明、博爱、男性、父亲、丈夫(女命)
- **性格**:豪爽、好施、积极、博爱、好动、急躁
- **庙陷分昼夜**:卯辰巳午为庙旺(日出至日中),戌亥子丑寅为落陷
- **吉**:化禄、化权、会六吉 → 大贵显达
- **凶**:化忌 → 男性长辈不利、易劳碌;落陷 → 与父、夫缘薄
- **入十二宫**:
  - 命宫:博爱光明,但易劳累
  - 夫妻(女命):夫君形象重要,落陷易生离克
  - 官禄:适合公职、公众事务
  - 父母:父亲影响深,落陷父先逝
- **庙旺**:卯辰巳午
- **落陷**:申酉戌亥子

### 4.4 武曲(财星/北斗第六)

- **五行**:辛金
- **属性**:刚毅、财富、孤克
- **性格**:刚直、寡言、决断、不善表达情感、勇敢
- **吉**:化禄(财运极佳);天府会 → 富格;化权 → 武贵
- **凶**:化忌 → 财来财去、健康问题(尤其呼吸、外伤);煞会 → 孤克婚姻
- **入十二宫**:
  - 命宫:个性刚毅、利财、不善交际
  - 夫妻:晚婚为吉,配偶刚毅,需有空间
  - 财帛:善理财、聚财
  - 官禄:武职、金融、工程
- **庙旺**:辰戌丑未(庙)、子午(平)
- **落陷**:无大陷,巳亥为平
- **女命**:武曲入命女子较硬朗,婚姻晚或多波折

### 4.5 天同(福星/南斗第四)

- **五行**:壬水
- **属性**:温和、福气、享受、儿童
- **性格**:温和善良、不争、随和、易满足、缺乏冲劲
- **吉**:六吉星会 → 福禄双全;天梁会 → 机月同梁格
- **凶**:化忌 → 福不全、易忧虑;煞会 → 失却安乐
- **入十二宫**:
  - 命宫:温和有福,但缺冲劲
  - 夫妻:配偶温和体贴
  - 福德:享受心强
- **庙旺**:巳亥(庙)、子午(旺)
- **落陷**:丑未(陷)

### 4.6 廉贞(次桃花/囚星/北斗第五)

- **五行**:丁火(含癸水)
- **属性**:双重性格(理性与感性兼具)、政治、桃花、刑罚
- **性格**:深沉、有主见、好胜、富心机、易感情用事、桃花重
- **吉**:化禄 → 偏财;天相会 → 福格;贪狼会 → 桃花福
- **凶**:化忌 → 感情纠纷、官非、血光;煞会 → 是非缠身
- **入十二宫**:
  - 命宫:多才多艺、感情复杂
  - 夫妻:感情起伏大,易有桃花
  - 官禄:政治、法律、公关
- **庙旺**:寅申(庙)
- **落陷**:巳亥(陷)
- **别称**:囚星(感情困囚)、次桃花(仅次于贪狼)

### 4.7 天府(库星/南斗主)

- **五行**:戊土
- **属性**:仓库、稳重、保守、聚财
- **性格**:温和稳重、保守、聚财守财、有度量、有些自负
- **吉**:辅弼会 → 富贵双全;武曲会 → 财富双库
- **凶**:空劫会 → 库破财空;独守无辅 → 略孤
- **入十二宫**:
  - 命宫:稳重保守、聚财能力强
  - 夫妻:配偶稳重持家
  - 财帛:稳定的财库
- **庙旺**:全宫皆吉,无落陷
- **特点**:天府为南斗主星,逢善则善、逢恶不恶,需配合辅佐看

### 4.8 太阴(母星/财星)

## 六、四化星

**四化**是紫微斗数最具特色的体系,由**生年天干**决定四颗主星(或辅星)分别化为**化禄、化权、化科、化忌**。

### 6.1 四化的意义

| 四化 | 含义 | 吉凶 |
|---|---|---|
| 化禄 | 财禄、好处、和谐、缘分 | 吉 |
| 化权 | 权力、能力、控制、强势 | 吉(过强则凶) |
| 化科 | 名声、贵人、学问、声誉 | 吉 |
| 化忌 | 阻碍、损失、纠缠、灾祸 | 凶(也有特殊吉用) |

### 6.2 生年四化表

| 生年干 | 化禄 | 化权 | 化科 | 化忌 |
|---|---|---|---|---|
| 甲 | 廉贞 | 破军 | 武曲 | 太阳 |
| 乙 | 天机 | 天梁 | 紫微 | 太阴 |
| 丙 | 天同 | 天机 | 文昌 | 廉贞 |
| 丁 | 太阴 | 天同 | 天机 | 巨门 |
| 戊 | 贪狼 | 太阴 | 右弼 | 天机 |
| 己 | 武曲 | 贪狼 | 天梁 | 文曲 |
| 庚 | 太阳 | 武曲 | 太阴 | 天同 |
| 辛 | 巨门 | 太阳 | 文曲 | 文昌 |

### 6.3 四化的运用

**生年四化**:基本盘永久存在,代表先天命格
**大限四化**:每十年根据大限宫的天干推四化,主导该10年
**流年四化**:每年根据流年干推四化,主导该年
**流月、流日四化**:同理细推

> 飞星派以**四化飞星**为核心,通过四化在十二宫间的流动,看事项发动。

### 6.4 化忌的特殊运用

- **忌入命宫**:本命有所执着、缺憾
- **忌入夫妻**:婚姻多波折
- **忌入财帛**:钱财不顺
- **忌冲对宫**:被冲之事受伤(冲迁移=外出不利、冲官禄=事业受阻)
- **忌入命宫为我执**:化忌之星即为命主一生执着之事
- **双忌**:大限/流年再化忌冲,大凶

## 七、三方四正

紫微斗数判命**不只看本宫**,必须连同三方四正合参。

### 7.1 定义

**本宫**:当前要看的宫位
**对宫**:本宫对面之宫(隔六宫)
**三合宫**:与本宫相隔四宫的两个宫(本宫之子辰申、寅午戌、巳酉丑、亥卯未三合)

**三方四正 = 本宫 + 对宫 + 两个三合宫 = 4个宫位**

### 7.2 三合宫表

| 本宫地支 | 三合一 | 三合二 |
|---|---|---|
| 子 | 申 | 辰 |
| 丑 | 巳 | 酉 |
| 寅 | 午 | 戌 |
| 卯 | 亥 | 未 |
| 辰 | 申 | 子 |
| 巳 | 酉 | 丑 |
| 午 | 戌 | 寅 |
| 未 | 亥 | 卯 |
| 申 | 子 | 辰 |
| 酉 | 丑 | 巳 |
| 戌 | 寅 | 午 |
| 亥 | 卯 | 未 |

### 7.3 六合(夹宫)

某宫**前后两宫**称为"夹宫",影响本宫(吉星夹则吉、煞星夹则凶)。

例:命宫在子,前宫为亥、后宫为丑,前后两宫的星曜以"夹拱"之力影响命宫。

### 7.4 实务运用

**举例**:命宫在午,看官禄运。
- **官禄宫**=戌(本宫)
- **对宫**=辰
- **三合宫**=寅、午

需综合看戌、辰、寅、午四宫的主辅星与四化,定事业运的整体格局。

## 八、大限 / 小限 / 流年

### 8.1 大限(10年一限)

**起讫岁数**:由五行局决定

| 五行局 | 起运岁 |
|---|---|
| 水二局 | 2岁 |
| 木三局 | 3岁 |
| 金四局 | 4岁 |
| 土五局 | 5岁 |
| 火六局 | 6岁 |

**起运方向**:阳男阴女顺行、阴男阳女逆行(同八字大运)

**例**:水二局阳年男命,命宫在午
- 第一大限:2—11岁(命宫,午)
- 第二大限:12—21岁(顺一,父母宫,未)
- 第三大限:22—31岁(顺二,福德宫,申)
- ...

**大限解读**:
- 大限宫之主星即此10年主调
- 大限四化:重要事项发动方向
- 大限三方四正:整体氛围

### 8.2 小限(1年一限)

小限以**生年支**起,**男顺女逆**,每年走一宫。

**起小限**:
- 寅午戌年生:小限从辰宫起1岁
- 申子辰年生:小限从戌宫起1岁
- 巳酉丑年生:小限从未宫起1岁
- 亥卯未年生:小限从丑宫起1岁

**例**:1990庚午年生男,小限从辰起1岁(男顺):
- 1岁辰、2岁巳、3岁午...

### 8.3 流年

流年以**当年地支**之宫为流年命宫。
流月、流日、流时:从流年命宫起算。

**实务**:小限+流年合参,流年权重>小限(现代多重流年)。

- **条件**:紫微+天府同宫(只在寅、申二宫可见)
- **意义**:帝王与库星同会,大富大贵

- **条件**:七杀、破军、贪狼在三方四正会照
- **意义**:剧烈变动、开创人生
- **应验**:经历起伏、有大成就也有大失败、创业家

- **条件**:天机、太阴、天同、天梁四星在三方四正会

- **条件**:太阳+天梁+文昌+禄存(或化禄)在三方四正

### 9.6 火贪 / 铃贪

- **条件**:火星+贪狼或铃星+贪狼同宫
- **意义**:暴发、武贵
- **应验**:突然致富、横财、武职峥嵘

> 《紫微斗数全书》:"贪狼遇火铃武职峥嵘,贪狼火星居庙旺为人勇猛威严。"

## 十一、案例分析

### 11.1 案例一:紫府同宫格

**命主**:女,1986年农历五月十二日午时生
**简略命盘**(纸面有限,只列关键):
- 命宫:申宫(紫微+天府,左辅+文昌)
- 财帛宫:辰宫(武曲+天相)
- 官禄宫:子宫(廉贞+破军)
- 夫妻宫:午宫(贪狼)

**生年丙年四化**:化禄=天同、化权=天机、化科=文昌、化忌=廉贞

**分析**:
1. **紫府同宫**于命宫,左辅+文昌为辅,**君臣庆会+紫府同宫**双格,先天命格极佳
2. **化科=文昌入命**,主聪明、学问、贵人
3. **官禄宫**廉贞化忌+破军 → 事业起伏大,易有官非或感情纠纷(廉贞化忌为感情之事)
4. **夫妻宫**贪狼独坐,桃花重,配偶有魅力但变动多;需注意感情与外遇
5. **武曲天相**于财帛宫 → 财库稳定,理财能力强

**大限走向**(假设水二局,2岁起运):
- 2—11岁(命宫,文教启蒙良好)
- 12—21岁(父母宫,长辈助力,学业有成)
- 22—31岁(福德宫,精神成长)
- 32—41岁(田宅宫,购房置业)
- ...

**结论**:富贵命,事业可有大成,但需注意感情管理(化忌+贪狼夫妻)与晚婚为佳。

### 11.2 案例二:杀破狼格

**命主**:男,1975年农历十月初八子时生
**简略命盘**:
- 命宫:亥宫(七杀,擎羊同宫)
- 财帛宫:未宫(贪狼)
- 官禄宫:卯宫(破军,化禄)
- 夫妻宫:酉宫(天相,化忌冲)

**生年乙年四化**:化禄=天机、化权=天梁、化科=紫微、化忌=太阴

**分析**:
1. **杀破狼**齐会三方,人生剧烈变动,起伏大
2. **七杀+擎羊**于命宫 → 个性刚烈、易有外伤、人生冲突多
3. **破军化禄**于官禄宫 → 事业开创力强,适合企业家、创业、变革性行业
4. **贪狼**于财帛宫 → 财从应酬、人际、技艺来
5. **天相**于夫妻宫,被太阴化忌之力影响(若三方有太阴) → 婚姻波折,需晚婚或异地姻缘
6. **化科=紫微**(假设在某宫拱命) → 有贵人、文化加持

**大限**:走七杀+擎羊宫位时,需特别注意伤灾、官非。

**结论**:开创型命格,适合创业、武职、改革性事业;早年波折,晚年有成;婚姻不顺,需经营。

## 十二、与八字对照

### 12.1 何时用紫微

**适合紫微的场景**:
- 看具体事项(婚姻、子女、健康、不动产等具体宫位)
- 看细节(谁帮你、谁损你)
- 看流年事件(配合飞星)
- 看人际关系(兄弟、朋友、上司)
- 看人格倾向(主星组合)

### 12.2 何时用八字

**适合八字的场景**:
- 看整体格局与人生主轴
- 看用神调候(寒暖燥湿)
- 看大运趋势
- 看健康五行偏颇
- 看财官印格局组合

### 12.3 两者合参

**最佳实践**:
1. 八字定**命主用神与大势走向**
2. 紫微看**具体事项细节与时间点**
3. 二者相互印证,准确率最高

**例如**:
- 八字看出某流年为大运中财运高峰
- 紫微看出该年财帛宫具体事项(投资、合作、横财)
- 综合判断:该年宜投资某行业、宜与某人合作

### 13.2 经典名言

> 《紫微斗数全书·总诀》:
> "夫紫微斗数者,源出陈希夷,自希夷传徐复斋,徐复斋传渔阳,渔阳传陈搏老祖。"
> 白话:紫微斗数源自陈希夷,代代相传至今。

> 《紫微斗数全书·骨髓赋》:
> "诸星吉,逢凶也吉;诸星凶,逢吉也凶。"
> 白话:吉星会照即使遇煞也减凶;凶星会照即使遇吉也增厄。

> 《紫微斗数全书·诸星问答》:
> "紫微帝座,在阳宫则吉,在阴宫则贪;
> 太阳化禄,在阳宫则贵,在阴宫则贱。"
> 白话:紫微在阳宫(子寅辰午申戌)为吉,在阴宫为贪;太阳在阳宫为贵,在阴宫(丑卯巳未酉亥)落陷为贱。

> 《紫微斗数全书·形性赋》:
> "原夫紫微帝座,生为厚重之容;
> 天府尊星,作为纯和之体。"
> 白话:紫微入命之人形貌厚重,天府入命之人体性纯和。
