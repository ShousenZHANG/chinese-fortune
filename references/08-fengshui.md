# 风水 (Feng Shui)

## 一、风水概述

风水，又称"堪舆"、"青乌"、"青囊"、"地理"，是中国古代研究人与自然环境关系的学问。其核心在于探究天地间"气"的运行规律，以求人居环境与天地气场的和谐共生。

### 源流与名称

风水之名最早见于晋代郭璞所著《葬书》：

> "气乘风则散，界水则止。古人聚之使不散，行之使有止，故谓之风水。"

"风水"二字，揭示其本质：

- **风**：气之载体与流动。
- **水**：气之凝聚与止泊。

风水之学，实为研究"气"如何被风带走、被水止住的方法论。气藏聚则吉，气散逸则凶。

## 三、主要流派

风水学派众多，大体分为"形势派"与"理气派"两大系统：

### 1. 形势派（峦头派）

注重山水形态、地形地貌，以**眼见为实**。代表为江西派，以杨筠松、曾文遄、廖瑀、赖文俊（"四大圣师"）为宗。

### 2. 理气派

注重罗盘运算、卦理、九宫、节气、星宿等抽象数理，以**推算为本**。代表为福建派。理气派下又分多支：

| 支派 | 核心 | 代表典籍 |
|------|------|---------|
| 八宅派 | 东西四宅、八卦本命 | 《八宅明镜》 |
| 玄空飞星 | 三元九运、山向飞星 | 《沈氏玄空学》 |
| 三元 | 上中下三元、卦运 | 《地理辨正》 |
| 三合 | 龙水向三合 | 《地理五诀》 |
| 玄空大卦 | 六十四卦、卦气 | 《地理辨正疏》 |
| 过路阴阳 | 八卦类象、应期 | 《金锁玉关》 |
| 金锁玉关 | 砂水定位 | 《过路阴阳》 |

### 形势 vs 理气之争

> "峦头无理气不灵，理气无峦头不验。" ——古谚

两派实为表里关系：形势重"形"，理气重"气"，形为气之载体，气为形之精神。

## 四、形势派核心：龙、穴、砂、水、向

### 1. 龙（山脉来势）

"龙"指山脉之走向起伏。山如龙之蜿蜒，故名。

- **来龙**：山脉从何而来。
- **过峡**：山脉中间的低凹处。
- **入首**：山脉入穴前的一段。

龙以**奔腾活动、形势磅礴**为吉；以**死板僵直、孤独瘦削**为凶。

### 2. 穴（结穴之地）

"穴"指气聚之点，乃天地灵气所钟。

- **正穴**：龙气尽止之处。
- **明堂**：穴前开阔之地。

穴有四种基本形态：

| 穴形 | 特征 |
|------|------|
| 窝穴 | 平地凹处，藏风聚气 |
| 钳穴 | 山脚分开如钳口 |
| 乳穴 | 山间凸起如乳房 |
| 突穴 | 平地突起 |

### 3. 砂（周边山）

"砂"指穴位四周护卫之山。古以四方圣兽喻之：

| 方位 | 圣兽 | 要求 |
|------|------|------|
| 东（左） | 青龙 | 蜿蜒、不可压迫 |
| 西（右） | 白虎 | 蹲伏、低于青龙 |
| 南（前） | 朱雀 | 朝拜有情 |
| 北（后） | 玄武 | 高大稳厚 |

> "青龙蜿蜒，白虎驯俯；玄武垂头，朱雀翔舞。" ——《葬书》

### 4. 水（水流）

"水"指穴位周围之水流。水为财之象。

- **来水**：水之源头处。
- **去水**：水之流出处。

水之吉凶：

| 水形 | 吉凶 |
|------|------|
| 玉带环腰（曲水绕宅） | 大吉 |
| 朝堂水（水流来朝） | 吉 |
| 反弓水（水曲背向） | 凶 |
| 冲心水（水直冲来） | 凶 |
| 割脚水（水紧贴宅基） | 凶 |
| 漏腮水（水从两侧斜出） | 凶 |

### 5. 向（坐山立向）

"向"指房屋或坟墓的朝向。坐山为屋后所靠之山，立向为屋前所朝之方。

定向之法：

- **坐北朝南**：传统正向，得阳光。
- **坐东朝西**：朝向夕阳。
- **坐西朝东**：朝向旭日。
- **坐南朝北**：背阳，少用。

## 六、罗盘基础

罗盘是风水师的核心工具，三盘并列：

### 三盘构成

| 盘 | 别名 | 用途 |
|----|------|------|
| 地盘正针 | 二十四山 | 定方位、定坐向 |
| 人盘中针 | 二十四山（错半位） | 消砂 |
| 天盘缝针 | 二十四山（错半位反向） | 纳水 |

- **正针**：磁针指示方向，对应地理方位。
- **中针**：比正针逆转 7.5°，用于格定砂山。
- **缝针**：比正针顺转 7.5°，用于格定水口。

### 三盘原理

杨筠松谓：

> "正针为天地之常，中针为人事之变，缝针为山川之实。"

## 十、阳宅内部布局

### 大门（纳气口）

> "宅以门为口，气从门入。"

大门是住宅之气的进口，最为关键：

- **大门朝向**：依宅卦命卦相配。
- **大门大小**：与宅之大小协调，不可过大或过小。
- **大门正对**：忌正对楼梯（漏财）、电梯（吸气）、卫生间（污秽）。

### 客厅（财位、聚气之所）

客厅为家庭活动中心，宜：

- **位于宅之中宫或前方**。
- **方正开阔**，不宜过分狭长。
- **采光良好**。
- **沙发后有靠**（实墙最佳）。

### 厨房（火门朝向）

厨房主财禄、健康：

- **厨房不宜位于宅之中心**（火气冲心）。
- **灶火门不宜对大门**。
- **灶不宜与卫生间相邻**（水火相战）。
- **灶不宜对窗**（财气外泄）。

灶位朝向通常按主人命卦定吉凶。

### 卧室（床位）

卧室关系睡眠、健康、夫妻关系：

- **床头有靠**（实墙、床头板）。
- **不宜对镜**（惊扰睡眠）。
- **不宜对门**（直冲不安）。
- **不宜在梁下**（压迫感）。
- **不宜对卫生间门**（湿气、秽气）。
- **不宜对厨房**（火气重）。

### 卫生间（不宜中宫）

卫生间为污秽之地：

- **不宜在宅之中宫**（污秽冲心）。
- **不宜对大门**（开门见厕，败财）。
- **不宜对厨房**（水火相战）。
- **保持干燥通风**。

### 书房（文昌位）

书房关系学业、事业文运：

- **位于文昌位最佳**。
- **书桌后有靠**。
- **书桌不宜对窗**（精神涣散）。
- **采光良好**。

### 神位 / 财位

- **神位**：宜位于宅内安静、洁净之处，背有靠，前有明堂。不宜对厕、对厨、对卧室门。
- **财位**：常见有"明财位"（入门斜对角）、"暗财位"（按宅卦推算）、"玄空财位"（按当令旺星方位）。

## 十一、财位判断

### 明财位

入户大门斜对角之处，最易判断。要点：

- 宜亮、宜净、宜安静。
- 不宜有窗（漏财）。
- 不宜杂物堆积。
- 可摆放绿植、寓意吉祥之物。

### 暗财位

按宅卦推算的财位，需专业人士判定。常用"八宅财位法"或"玄空财位法"。

### 玄空财位

按当运旺星所在方位，例如八运（2004-2023）八白当令，八白星所到之宫位即为旺财位；九运（2024-2043）九紫当令，九紫所到为旺财位。

## 十二、文昌位

文昌位主学业、考试、写作、思维清晰。判断方法：

### 流年文昌

每年文昌位不同，按四绿文昌星飞临之宫位定。

### 宅卦文昌

按宅之坐山推算：

| 坐山 | 文昌位 |
|------|--------|
| 坎宅（坐北朝南） | 东北 |
| 离宅（坐南朝北） | 南 |
| 震宅（坐东朝西） | 西北 |
| 兑宅（坐西朝东） | 西南 |
| 巽宅（坐东南朝西北） | 中宫 |
| 乾宅（坐西北朝东南） | 东 |
| 坤宅（坐西南朝东北） | 西 |
| 艮宅（坐东北朝西南） | 北 |

### 个人文昌

按本人出生年命，结合八字日干。如：

- 甲日生：文昌在巳。
- 乙日生：文昌在午。
- 丙、戊日生：文昌在申。
- 丁、己日生：文昌在酉。
- 庚日生：文昌在亥。
- 辛日生：文昌在子。
- 壬日生：文昌在寅。
- 癸日生：文昌在卯。

## 十三、桃花位

桃花位主感情、人缘、社交：

### 流年桃花

按流年九星飞布，一白星或当年桃花飞临之宫为桃花位。

### 命理桃花

依年支或日支判定：

## 十四、化煞方法（现代理性化）

风水化煞之法繁多。本节倾向推荐以"调整布局、环境心理学、建筑学常识"为主的现代化解之法，避免迷信化的产品依赖。

### 1. 调整布局

- **重新规划家具位置**：床位、办公位避开冲煞方向。
- **改变门向**：必要时调整内门走向。
- **隔断处理**：玄关、屏风分隔气场。

### 2. 绿植调节

- **生气位**：绿植旺财气。
- **凶位**：耐阴植物可吸纳浊气。
- **避免**：尖叶植物（仙人掌等）放置于卧室、客厅显眼处。

### 3. 光线调节

- **采光充足**：阳气足，气场清明。
- **暗角补光**：消除阴气滞留。
- **避免强烈直射**：眩光、热气过盛亦非吉。

### 4. 镜子用法与禁忌

镜子在风水中作用极为微妙：

| 宜 | 忌 |
|----|----|
| 玄关一侧反射光线 | 卧室对床 |
| 暗处反射自然光 | 大门正对 |
| 化解某些煞气 | 厨房对灶 |
| | 镜片碎裂久置 |

### 5. 颜色搭配

颜色对应五行：

| 颜色 | 五行 |
|------|------|
| 红、橙、紫 | 火 |
| 黄、棕、土黄 | 土 |
| 白、金、银 | 金 |
| 黑、深蓝 | 水 |
| 绿、青 | 木 |

按宅卦、个人命卦五行喜忌，调整主色调。例：木命人宜绿色基调，火命人宜红色基调。

## 十五、办公室风水

### 老板办公室

- **位置**：宜位于公司之西北（乾位，主权威）或后方。
- **办公桌**：背墙不背窗，正面有适当开阔空间。
- **门向**：避开冲门、走道直冲。

### 员工办公位

- **避免**：背门（不安全感）、面壁、走道边。
- **宜**：背墙、面向门口、采光良好。

### 财务部门

- **位置**：宜近老板办公室，方位选当令旺星方。
- **保险柜朝向**：宜向内、宜隐蔽。

## 十六、商铺风水

### 店面选址

- **气口**：人流量、路冲、商圈成熟度。
- **形势**：宜门面开阔，迎宾纳客。
- **避忌**：店门对楼梯下行（财向下）、店面后有压顶。

### 店内布局

- **收银台**：位于店内的财位（白虎位或当令旺星方）。
- **货架**：流通顺畅，不阻塞主气路。
- **门口**：明亮整洁，便于纳气。

### 招牌

- **位置高度**：适中，不可过高（飘）或过低（压）。
- **颜色**：与行业五行相配。
  - 餐饮（火）：红、橙。
  - 金融（金）：白、银、金。
  - 教育（木）：绿、蓝。
  - 建筑（土）：黄、棕。
  - 海洋、运输（水）：黑、蓝。

## 十七、阴宅基础（文化介绍）

阴宅风水涉及法律、伦理与文化习俗，本 reference 仅作文化层面的简要介绍，不展开实操指导。

### 阴宅核心原则

古代阴宅讲究"葬乘生气"（《葬书》语），追求三个目标：

1. **避煞**：避开形势、地质、水文之凶。
2. **得气**：寻找龙脉聚气之处。
3. **荫后**：以求福荫后世子孙。

### 文化背景

阴宅风水在中国传统文化中具有重要意义：

- **慎终追远**：体现儒家"孝"的价值观。
- **天人感应**：相信生者与逝者气脉相通。
- **家族延续**：以阴宅福荫后代。

### 现代视角

现代社会，阴宅风水多在以下层面延续：

- **公墓选择**：参考地势、朝向、环境。
- **风俗仪式**：保持传统礼仪的文化价值。
- **心理慰藉**：让生者获得情感安宁。

注：现代阴宅事宜须遵守国家殡葬法规与土地政策，建议以公墓制式安葬为主。

## 十八、现代风水适用性

### 心理学解读

风水中很多禁忌可用环境心理学解释：

- **床不对门**：减少入睡时的"被看见"焦虑感。
- **床头有靠**：增强睡眠安全感。
- **大门见绿**：减压、舒缓情绪。
- **客厅明亮**：提升积极情绪。
- **卧室宁静**：促进睡眠质量。

### 环境学解读

许多"风水原则"实为环境工学常识：

- **采光通风**：影响健康、心情。
- **湿度温度**：影响舒适度与寿命。
- **噪音污染**：影响睡眠、神经。
- **空气流通**：影响呼吸系统。
- **绿植配置**：净化空气、调节湿度。

### 理性建议

将风水视为"环境优化+文化心理"的综合学问：

- 吸纳其中合理部分（朝向、采光、布局）。
- 摈弃迷信化、消费化的部分。
- 不必依赖购买高价"开光物品"。
- 调整布局与心态，方为根本。

## 二十、学习路径

### 入门阶段

1. **基础理论**：阴阳、五行、八卦、十干十二支。
2. **形势观察**：识山、识水、识方位。
3. **罗盘使用**：基本格法、二十四山。

### 进阶阶段

1. **流派选择**：选定一派深入，如八宅或玄空。
2. **典籍研读**：精读《葬书》《沈氏玄空学》等。
3. **案例实践**：实地考察名宅、古墓。

### 精通阶段

1. **多派融通**：理气与形势并重。
2. **现代结合**：与建筑学、环境学、心理学融贯。
3. **实战检验**：长期跟踪验证。

## 二十一、风水的现代价值

风水之学，源于古人对人居环境的精细观察，蕴含建筑学、环境学、心理学、人文学多重智慧。其精华在于：

1. **天人合一的整体观**：人与自然不可分离。
2. **形与气并重的方法论**：可见与不可见之统一。
3. **趋吉避凶的实用性**：优化环境、改善生活。
4. **文化传承的延续性**：维系传统价值。

> "风水之道，存乎一心；心正则气正，气正则地正。" ——古谚

学者宜以理性态度对待，取其精华、去其糟粕。风水不应成为牟利工具，亦不应被神秘化。在现代社会，风水更多作为一种"环境优化+心理调适+文化传承"的综合智慧，发挥其应有的价值。
