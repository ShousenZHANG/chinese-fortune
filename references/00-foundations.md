## 1. 阴阳

> 《易传》："一阴一阳之谓道。"

阴阳是中国哲学最底层的二元对立统一。万物皆可分阴阳，阴阳互根、互转、消长、平衡。

| 阳 | 阴 |
|---|---|
| 天 | 地 |
| 日 | 月 |
| 男 | 女 |
| 动 | 静 |
| 刚 | 柔 |
| 进 | 退 |
| 升 | 降 |
| 外 | 内 |
| 奇数 | 偶数 |
| 白昼 | 夜晚 |

**核心原理**：
- **互根**：孤阴不生，独阳不长。
- **消长**：日中则昃，月盈则亏。
- **转化**：物极必反，重阴必阳，重阳必阴。

## 2. 五行

五种基本能量形态：**木 · 火 · 土 · 金 · 水**。

### 生克

| 关系 | 顺序 | 口诀 |
|---|---|---|
| 相生 | 木→火→土→金→水→木 | 木生火、火生土、土生金、金生水、水生木 |
| 相克 | 木→土→水→火→金→木 | 木克土、土克水、水克火、火克金、金克木 |
| 相泄 | 我生者泄我 | 如金生水，水泄金 |
| 相耗 | 我克者耗我 | 如金克木，木耗金 |
| 相侮 | 反克 | 弱者反克强者 |

### 属性映射

| 五行 | 方位 | 季节 | 颜色 | 味道 | 脏 | 腑 | 情志 | 数 |
|---|---|---|---|---|---|---|---|---|
| 木 | 东 | 春 | 青/绿 | 酸 | 肝 | 胆 | 怒 | 3·8 |
| 火 | 南 | 夏 | 红 | 苦 | 心 | 小肠 | 喜 | 2·7 |
| 土 | 中 | 长夏 | 黄 | 甘 | 脾 | 胃 | 思 | 5·10 |
| 金 | 西 | 秋 | 白 | 辛 | 肺 | 大肠 | 悲 | 4·9 |
| 水 | 北 | 冬 | 黑 | 咸 | 肾 | 膀胱 | 恐 | 1·6 |

### 旺相休囚死

每个五行在不同季节强度不同：

| 五行 | 旺 (当令) | 相 (生之) | 休 (生我) | 囚 (克我) | 死 (我克) |
|---|---|---|---|---|---|
| 木 | 春 | 冬 | 夏 | 秋 | 长夏 |
| 火 | 夏 | 春 | 长夏 | 冬 | 秋 |
| 土 | 长夏 | 夏 | 秋 | 春 | 冬 |
| 金 | 秋 | 长夏 | 冬 | 夏 | 春 |
| 水 | 冬 | 秋 | 春 | 长夏 | 夏 |

## 7. 24 节气

| 月支 | 节 | 气 |
|---|---|---|
| 寅 (1月) | 立春 | 雨水 |
| 卯 (2月) | 惊蛰 | 春分 |
| 辰 (3月) | 清明 | 谷雨 |
| 巳 (4月) | 立夏 | 小满 |
| 午 (5月) | 芒种 | 夏至 |
| 未 (6月) | 小暑 | 大暑 |
| 申 (7月) | 立秋 | 处暑 |
| 酉 (8月) | 白露 | 秋分 |
| 戌 (9月) | 寒露 | 霜降 |

## 9. 十神 (10 Gods)

以日干为"我"，其余天干（及地支藏干）与我的关系定义为十神：

| 十神 | 阴阳关系 | 含义 |
|---|---|---|
| 比肩 (比) | 同我同性 | 兄弟、朋友、自我 |
| 劫财 (劫) | 同我异性 | 异性朋友、争夺、合伙 |
| 食神 (食) | 我生同性 | 才艺、享受、子女(女命) |
| 伤官 (伤) | 我生异性 | 表达、叛逆、子女(女命) |
| 偏财 (才) | 我克同性 | 偏门财、外财、父亲 |
| 正财 (财) | 我克异性 | 正职财、妻 (男命) |
| 七杀 (杀) | 克我同性 | 压力、敌人、儿 (男命，亦称偏官) |
| 正官 (官) | 克我异性 | 工作、夫 (女命)、地位 |
| 偏印 (枭) | 生我同性 | 偏才、继母、宗教 |
| 正印 (印) | 生我异性 | 学业、母、靠山、文书 |

**示例**：日干 = 甲木 (阳木)
- 乙 = 劫财 (阴木)
- 丙 = 食神 (阳火，我生同性)
- 丁 = 伤官 (阴火)
- 戊 = 偏财 (阳土，我克同性)
- 己 = 正财 (阴土)
- 庚 = 七杀 (阳金)
- 辛 = 正官 (阴金)
- 壬 = 偏印 (阳水)
- 癸 = 正印 (阴水)
